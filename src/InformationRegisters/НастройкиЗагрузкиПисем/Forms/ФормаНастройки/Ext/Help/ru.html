Здесь можно настроить вариант загрузки электронных писем  и вариант синхронизации признаков "Просмотр" и "Удаление" для каждой папки на сервере почтовой службы IMAP.

По команде Обновить почтовые папки УНФ перечитывает почтовые папки с почтового сервера.

Колонка Вариант загрузки:

  • Не загружать — письма из указанной папки не будут загружаться,
  • Только заголовки — из указанной папки будут загружаться только заголовки писем,
  • Тексты и вложения — из указанной папки будут загружаться тексты и вложения.


Колонка Вариант синхронизации:

  • Не синхронизировать — признаки "Просмотр" и "Удаление" не синхронизируются,
  • Просмотр — при каждой загрузке писем просмотренные в УНФ письма отмечаются как просмотренные на почтовом сервере,
  • Удаление — при каждой загрузке писем помеченные на удаление в УНФ письма удалются с почтового серера,
  • Просмотр и удаление – при каждой загрузке выполняется синхронизация обоих признаков.
