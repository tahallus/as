Как находясь на рабочем месте по работе с ККМ Offline посмотреть список товаров загружаемый на ККМ?

Для этих целей используется команда списка обмена "Посмотреть список товаров".

Какие действия можно выполнить в списке товаров оборудования?

В форме просмотра можно установить отбор по товарам: "Все", "Измененные" (устанавливается при открытии формы по умолчанию), "С ошибками" (без цены и/или штрих-кода). Так же можно выгрузить измененные товары в ККМ/весы, очистить и полностью перезагрузить ККМ/весы, распечатать список товаров с кодами ККМ/PLU, распечатать ценники выгружаемых товаров.
