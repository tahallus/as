Настройка передачи товаров между организациями

Рабочее место предназначено для настройки схемы совместной работы организаций на предприятии (схемы Интеркампани). В этом случае при продаже товаров клиенту будет автоматически регистрироваться информация о том, что от имени одной организации был продан товар другой организации. В результате на основании этих данных будут автоматически формироваться документы Передача товаров между организациями.

Использование схемы межфирменных продаж между организациями (Интеркампани) возможно только при включенных функциональных опциях Передачи товаров между организациями в разделе Еще больше возможностей – Компания.

Доступны следующие основные варианты передачи товаров:

  • Передача через внутреннюю продажу. Организация может продавать товары организации-владельца. Передача товаров между организациями будет оформляться с помощью операции купли-продажи между организациями.
  • Передача товаров не выполняется. Организация не может продавать товары организации-владельца, формирование резервов передачи между этими организациями выполняться не будет.

Для исключения организации из схемы межфирменных продаж достаточно установить для данной организации значение способа передачи данных – Не требуется передачи товаров. Исключить организацию из схемы межфирменных продаж можно в любой момент времени. При этом в программе сохранятся все данные об оформленных ранее продажах данной организации в схеме межфирменных продаж, а новые продажи по схеме межфирменных продаж регистрироваться не будут.
