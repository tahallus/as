Регистр сведений "Штатное расписание"

Регистр предназначен для хранения актуальной информации о штатном расписании организации.

Для каждой записи штатного расписания необходимо указать следующие реквизиты:

  • Организация, для которой вводится запись штатного расписания. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Подразделение, для которого вводится запись штатного расписания. Выбирается из справочника "Структурные единицы".
  • Должность, для которой вводится запись штатного расписания. Выбирается из справочника "Должности".
  • Количество ставок, которое планируется в выбранных организации и подразделении на указанной должности.
  • Вид начисления (удержания), по которому планируется оплачивать работу сотрудников. Выбирается из списка документов "Виды начислений и удержаний".
  • Минимальная и Максимальная тарифные ставки, которые планируются для единицы штатного расписания.
  • Валюта тарифной ставки. Выбирается из справочника "Валюты".
