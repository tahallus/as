Предназначена для настройки параметров подключения мобильного приложения "Проверка ценников".

Открывается по команде Настроить синхронизацию параграфа 1С:Проверка ценников формы Каталог мобильных приложений из меню  Настройки - Интеграции.

С помощью данной формы указать параметры подключения мобильного приложения и сформировать QR-код с выбранными настройками для его считывания в мобильном приложении 1С: Проверка Ценников:

  • В поле Организация необходимо выбрать организацию, от имени которой производится обмен; 
  • Рабочее место позволяет справочно указать место создания настройки мобильного приложения;
  • Правило обмена содержит методику отбора выгружаемой номенклатуры;
  • Количество элементов в пакете отражает методику передачи перечня отобранной номенклатуры;
  • Идентификатор устройства позволяет установить соответствие с конкретным экземпляром мобильного приложения;
  • Пользователь приложения указывает на учётную запись от имени которой будет осуществляться подключение.
