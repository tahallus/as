Форма настройки обработчика драйвера "Терминалы сбора данных"

Форма предназначена для настройки параметров работы терминалов сбора данных. Форма позволяет настраивать следующие параметры устройства:

  • Модель
    Модель подключаемого устройства.
  • Порт
    Порт, к которому подключен терминал сбора данных.
  • Скорость
    Скорость обмена данных с терминалом.
  • Разделитель документа
    Символ или код символа, используемые в качестве разделителя полей в документах терминала сбора данных.
  • Таблица выгрузки
    Таблица выгрузки терминала, в которую происходит выгрузка.
  • Таблица загрузки
    Таблица загрузки терминала сбора данных, из которого происходит загрузка.

После изменения значений параметров следует нажать кнопку "ОК" для того, чтобы сохранить внесённые изменения, или кнопку "Отмена" для того, чтобы отказаться от их сохранения.
