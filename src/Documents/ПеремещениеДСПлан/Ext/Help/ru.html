Документ "Перемещение денег (план)"

Документ "Перемещение денег (план)" предназначен для планирования перемещения денежных средств.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется планирование. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Валюта. Указывается валюта планирования. Выбирается из справочника "Валюты".
  • Сумма. Указывается планируемая сумма платежа в валюте планирования.
  • Статья движения денежных средств, по которой планируется перемещение денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Статус платежа. Указывается одним из вариантов: утвержден или не утвержден.
  • Источник денежных средств. Группа реквизитов:
    • Тип денежных средств источника. Указывается один из вариантов: безналичные или наличные.
    • Банковский счет источника организации, для которой планируется перемещение. Становится доступным для указания при выбранном типе денежных средств - безналичные. Выбирается из справочника "Банковские счета".
    • Касса источника организации, для которой планируется перемещение. Становится доступным для указания при выбранном типе денежных средств - наличные. Выбирается из справочника "Кассы".
  • Получатель денежных средств. Группа реквизитов:
    • Тип денежных средств получателя. Указывается один из вариантов: безналичные или наличные.
    • Банковский счет получателя организации, для которой планируется перемещение. Становится доступным для указания при выбранном типе денежных средств - безналичные. Выбирается из справочника "Банковские счета".
    • Касса получателя организации, для которой планируется перемещение. Становится доступным для указания при выбранном типе денежных средств - наличные. Выбирается из справочника "Кассы".
  • Комментарий. Указывается комментарий к документу.
