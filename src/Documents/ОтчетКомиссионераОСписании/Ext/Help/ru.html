Документ "Отчет комиссионера о списании"

Документ предназначен для проведения списания товаров переданных на комиссию.


Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Комиссионер. В данном реквизите указывается информация о комиссионере. Выбирается из справочника "Контрагенты". 

  • Договор. В данном реквизите указывается договор с комиссионером, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".

  • Основание. В данном реквизите указывается "Расходная накладная", на основании которой сформирован данный документ. 

  • Заказ. В данном реквизите указывается заказ, по которому произошла передача товаров на комиссию. 


На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования валюты и вида цен данного документа. Может быть установлен флаг Регистрировать цены поставщика, в этом случае цены будут записаны для дальнейшего анализа и подстановки в документы.


В подвале формы можно указать ручную скидку на документ, процентом или суммой.

  • Товары. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, поступление которой отражается данным документом. Выбирается из справочника "Номенклатура" с типом "Запас". 

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит обязателен для заполнения, если вид операции документа: "Прием на комиссию", "Прием в переработку" или "Прием на ответственное хранение". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество поступившей номенклатурной позиции.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Цена - цена номенклатурной позиции.

    • Сумма- стоимость номенклатуры по данной строке.

    • Заказ - указывается заказ покупателя, по которому была передача на комиссию.


Минимальные цены


При включении опции "Минимальные цены продажи" и настройки минимальных цен (Раздел "Продажи" -> "Еще больше возможностей") в табличной части цвет текста поля цена будет изменен на красный, если значение цены станет меньше чем минимальная.
Минимальные цены будут подбираться из общего вида минимальных цен.
