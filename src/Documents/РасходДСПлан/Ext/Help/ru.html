Документ "Заявка на расход денег"

Документ "Заявка на расход денег" предназначен для планирования расходования денежных средств.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется планирование. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Валюта. Указывается валюта планирования. Выбирается из справочника "Валюты".
  • Сумма. Указывается планируемая сумма платежа в валюте планирования.
  • Статья движения денежных средств, по которой планируется расход денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Статус платежа. Указывается один из вариантов: утвержден или не утвержден.
  • Способ оплаты. Указывается один из вариантов: безналичные, наличные или не определен.
  • Банковский счет организации, для которой планируется расход. Становится доступным для указания при выбранном способе оплаты - безналичные. Выбирается из справочника "Банковские счета".
  • Касса организации, для которой планируется расход. Становится доступным для указания при выбранном способе оплаты - наличные. Выбирается из справочника "Кассы".
  • Документ-основание. Группа реквизитов:
    • Вх. номер. Номер входящего документа-основания для платежа.
    • От. Дата входящего документа-основания для платежа. 
    • Документ-основание планирования расхода.
  • Комментарий. Указывается комментарий к документу.
