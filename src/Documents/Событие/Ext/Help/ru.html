Документ "Событие"

Документ предназначен для отправки электронной почты и регистрации событий пользователем.

 

При создании нового события необходимо указать тип события. От типа события зависит состав заполняемых реквизитов.

 

Для типа события "Электронное письмо" заполняются следующие реквизиты:

  • Контакты. Данная табличная часть содержит перечень лиц, которым осуществляется рассылка. Реквизиты табличной части:

    • Контакт - участник рассылки. Возможно заполнение строкой или выбором из справочников "Контрагенты", "Контактные лица". При нажатии на кнопку выбора открывается форма адресной книги.

    • Адрес электронной почты - адрес рассылки. Возможно указание нескольких адресов через разделитель [;].

  • Вложения. Данная табличная часть содержит список отправляемых вложений. Добавление новых вложений осуществляется по контекстному меню или перетаскиванием файлов из операционной системы. При отправке письма вложения сохраняются в виде присоединенных к документу файлов.

  • Тема. Тема письма.

  • Содержание. Тело письма.

  • Учетная запись. Учетная запись электронной почты с которой будет осуществляться отправка.

  • Состояние. В случае успешной отправки письма автоматически устанавливается состояние "Завершено".

Для остальных типов событий заполняются следующие реквизиты:

  • Состояние. В данном реквизите указывается категория классификации события. Выбирается из справочника "Состояния событий". Реквизит обязателен для заполнения.

  • Контрагент. В данном реквизите указывается информация о контрагенте. Выбирается из справочника "Контрагенты". Реквизит необязателен для заполнения.

  • Тема. В данном реквизите указывается основное описание события. Реквизит необязателен для заполнения.

  • Содержание. В данном реквизите указывается описание события. Реквизит необязателен для заполнения.

  • Документ основание. В данном реквизите указывается основание события. Реквизит необязателен для заполнения.

  • Начало события. В данном реквизите указывается дата и время начало события. Реквизит необязателен для заполнения.

  • Окончание события. В данном реквизите указывается дата и время окончания события. Реквизит необязателен для заполнения.

  • Ответственный. В данном реквизите указывается информация о сотруднике, ответственном за ввод текущего документа. Выбирается из справочника "Сотрудники".  Реквизит необязателен для заполнения.

  • Участники. Данная табличная часть содержит перечень участников события. Реквизиты табличной части:

    • Контакт - участник встречи. Возможно заполнение строкой или выбором из справочников "Контрагенты", "Контактные лица".

    • Как связаться - контактная информация участника.

 

В группе "Дополнительные реквизиты" указываются дополнительные реквизиты, определяемые пользователем.
