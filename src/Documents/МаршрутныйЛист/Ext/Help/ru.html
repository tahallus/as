Документ "Маршрутный лист"

Документ предназначен для отражения факта передачи заказов курьеру или службе доставки..

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид доставки. Указывается один из видов доставки: собственным курьером или с помощью сторонней службы доставки
  • Служба доставки. В данном реквизите указывается сторонняя служба доставки. Выбирается из справочника "Службы доставки". При выборе вида доставки "Курьером" реквизит не отображается на форме и заполняется автоматически предопределенной службой доставки "Доставка собственными силами".

  • Способ отгрузки. В данном реквизите задается способ отгрузки заказов. Доступен только при доставке службой.

  • Склад отгрузки. В данном реквизите указывается склад, с которого будет выполняться отгрузка заказов. Выбирается из справочника "Структурные единицы" с типом "Склад". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким складам и используются способы отгрузки "Передать курьеру службы доставки" или "Передать курьеру единого склада".

  • Адрес отгрузки. Реквизит не редактируется напрямую, а заполняется фактическим адресом склада при ведении учета по нескольким складам или фактическим адресом организации в других случаях.

  • Курьер. В данном реквизите указывается курьер, который будет выполнять заказ. Выбирается из справочника "Физические лица". Отображается на форме при использовании способа доставки "Курьером".

 

Для корректной работы документа следует задать состояния доставки заказов покупателей для каждого вида заказов. Форма установки состояний откроется автоматически при вводе первого документа. В дальнейшем состояния можно изменить используя ссылки на форме маршрутного листа.

Под реквизитами шапки расположен список доступных к отгрузке заказов и сворачиваемая группа отборов. Для добавления заказа в маршрутный лист следует установить отметку напротив него. Доступно также исключение заказов из маршрутного листа снятием отметки, но только до момента формирования документов отгрузки.

Доступно массовое формирование и печать расходных накладных для отгружаемых заказов.
