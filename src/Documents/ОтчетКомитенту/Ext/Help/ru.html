Документ "Отчет комитенту"

Документ предназначен для хранения информации об отчетах перед комитентом за реализованные запасы, принятые от него на комиссию, и для расчета комиссионного вознаграждения.

 

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, которой передавались запасы на комиссию от комитента. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Контрагент. В данном реквизите указывается комитент. Выбирается из справочника "Контрагенты".

  • Договор. В данном реквизите указывается договор контрагента, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".

 

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования видов цен и валюты данного документа.

  • Дополнительно. Группа реквизитов:

    • Способ расчета комиссионного вознаграждения. Указывается способ расчета комиссионного вознаграждения. Выбирается из следующих вариантов:

      • Не рассчитывается (такой способ расчета означает, что сумма комиссионного вознаграждения указывается индивидуально).

      • Процент от суммы продажи (под суммой продаж понимается та сумма, по которой организация продала запасы).

      • Процент от разности сумм продажи и поступления (под суммой продаж понимается та сумма, по которой организация продала запасы, а под суммой поступления понимается та сумма, по которой запасы были приняты на комиссию).

    • Процент комиссионного вознаграждения. Указывается процент для расчета комиссионного вознаграждения.

    • Ставка НДС с комиссионного вознаграждения. Указывается ставка налога на добавленную стоимость с комиссионного вознаграждения.

    • Удержать комиссионное вознаграждение. Указывается признак удержания комиссионного вознаграждения.

  • Запасы. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, за реализацию которой организация отчитывается перед комитентом. Выбирается из справочника "Номенклатура" с типом "Запас".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - партия номенклатуры, по которой номенклатурная позиция принималась на комиссию. Выбирается из справочника "Партии номенклатуры".

    • Количество - количество реализованной номенклатуры, за которую отчитывается организация.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Цена - цена, по которой номенклатурная позиция была реализована организацией.

    • Сумма - стоимость номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - поле, автоматически заполняемое суммарной стоимостью запасов, которые были реализованы организацией, согласно настройкам цен и валют. Настройки устанавливаются в диалоге, вызываемом по гиперссылке "Редактировать цены и валюту".

    • Страна происхождения - страна происхождения запасов (см. опцию "Прямое списание ГТД").

    • Номер ГТД - грузовая таможенная декларация запаса (см. опцию "Прямое списание ГТД").

    • Прямое списание ГТД - указывает способ отражения движений по номерам ГТД. Если опция включена, то программа спишет остаток по номеру ГТД из собственных запасов, а если выключена списание номера ГТД произойдет из запасов переданных.

    • Цена поступления - цена, по которой номенклатурная позиция была принята на комиссию.

    • Сумма поступления - стоимость, по которой номенклатурная позиция была принята на комиссию. 

    • Сумма НДС поступления - сумма налога на добавленную стоимость с суммы поступления по данной строке.

    • Сумма вознаграждения - сумма комиссионного вознаграждения, рассчитанная исходя из условий договора с комитентом и параметров закладки "Дополнительно".

    • Сумма НДС вознаграждения - сумма налога на добавленную стоимость с суммы вознаграждения по данной строке.

    • Заказ поступления - заказ поставщику, в рамках которого принимались запасы на комиссию. Выбирается документ "Заказ поставщику". Реквизит необязателен для заполнения.

    • Заказ покупателя - заказ покупателя, в рамках которого реализованы запасы, принятые на комиссию организацией. Выбирается документ "Заказ покупателя". Реквизит необязателен для заполнения.

    • Покупатель запасов комитента. Значение реквизита используется в печатной форме "Отчет комитенту".

    • Дата реализации покупателю запасов комитента. Значение реквизита используется в печатной форме "Отчет комитенту".

  • Предоплата. Данная табличная часть заполняется сведениями по зачету авансов комитенту. На форме документа расположена гиперссылка "Редактировать зачет предоплаты", по которой вызывается диалог редактирования табличной части "Зачет предоплаты". Реквизиты табличной части:

    • Документ - документ поступления аванса от покупателя: "Расход из кассы", "Расходы со счета".

    • Заказ - документ "Заказ поставщику".

    • Сумма расчетов - сумма в валюте договора.

    • Курс - курс пересчета в валюту документа.

    • Кратность - кратность пересчета в валюту документа.

    • Сумма платежа - указывается в валюте документа.

  • Дополнительная информация. Группа реквизитов:

    • Подразделение - информация о подразделении, ответственном за операцию. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение". Реквизит необязателен для заполнения.

    • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.
