Документ "Задание на работу"

Документ предназначен для планирования занятости сотрудника. 

 

В зависимости от установленного вида операции с помощью данного документа могут быть запланированы:

  • работа по какому-либо виду работ по конкретному заказу, по договору, для контрагента (Внешнее задание);

  • работа по какому-либо виду работ для компании (Внутреннее задание).

    • В документе необходимо заполнить следующие реквизиты:

    • Организация, от лица которой оформляется задание. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

    • Состояние. В данном реквизите указывается категория классификации задания. Выбирается из справочника "Состояния событий". Реквизит обязателен для заполнения.

    • Вид операции. Необходимо выбрать вид задания: Внешнее (выполнение какого-либо задания по конкретному заказу, по договору), Внутренняя (выполнение какого-либо задания для компании).
    • Подразделение, в котором планируется задание. Выбирается из справочника "Структурные единицы".
    • Сотрудник, которому ставится задание. Выбирается из справочника "Сотрудники".
    • Заказчик, для которого планируется выполнение задания. Выбирается из справочника "Контрагенты", "Договоры контрагентов", "Заказы покупателей".
    • Работа (услуга), для изготовления которой выполняется задание. Выбирается из справочника "Номенклатура".
    • Характеристика работы (услуги), для изготовления которой выполняется задание. Выбирается из справочника "Характеристики номенклатуры".
    • Вид работ, деятельность по которому планируется. Выбирается из справочника "Номенклатура".
    • День - планируемая дата выполнения задания.
    • Время начала выполнения задания.
    • Время окончания выполнения задания.
    • Длительность выполнения задания в формате ЧЧ:ММ.
    • Длительность (ч) - длительность выполнения задания в часах.
    • Вид цен, по которому определяется стоимость работ для заказчика. Выбирается из справочника "Виды цен".
    • Цена - стоимость часа работы по заданию.
    • Сумма - стоимость выполнения задания для клиента.
    • Описание. Указывается любая дополнительная информация к документу.
     
    Минимальные цены

    При включении опции "Минимальные цены продажи" и настройки минимальных цен (Раздел "Продажи" -> "Еще больше возможностей") в табличной части цвет текста поля цена будет изменен на красный, если значение цены станет меньше чем минимальная.
    Минимальные цены будут подбираться из общего вида минимальных цен.

    На основании документа "Задание на работу" может быть введен документ "Учет времени".
