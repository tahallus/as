Документ "Комплектация"

Документ предназначен для отражения единичной операции комплектации товаров на складе.

С помощью данного документа могут быть зафиксированы следующие операции:

  • Сборка - комплектация товаров на складе. В шапке указывается собираемый запас. В табличной части "Товары" указываются израсходованные комплектующие. 
  • Разборка - разборка исходного запаса. В шапке указывается исходная номенклатура, которую необходимо разобрать. В табличной части "Товары" указывается выходной продукт, т.е. результат разборки.

Необходимо заполнить следующие реквизиты: 

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ.
  • Вид операции. Вид операции документа.
  • Склад. В этом поле необходимо указать структурную единицу, из которой будут списаны запасы. В отличие от документов производства, комплектация может выполняться на розничных складах без предварительного перемещения.
  • Ячейка. В этом поле необходимо указать ячейку, из которой будут списаны запасы.
  • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
  • Характеристика. При необходимости в данном поле указывается характеристика номенклатурной позиции. Поле доступно только в том случае, если в настройках учета указано ведение учета по характеристикам номенклатуры. Выбирается из справочника "Характеристики номенклатуры".
  • Партия. Указывается партия номенклатурной позиции. Поле доступно только в том случае, если в настройках учета указано ведение учета по партиям номенклатуры. Выбирается из справочника "Партии". 
  • Ед. Единица измерения, в которой вводится количество номенклатуры. Поле доступно только в том случае, если в настройках включен учет в нескольких единицах измерения.
  • Количество. Указывается количество единиц измерения номенклатурной позиции. 
  • Страна происхождения номенклатурной позиции. Выбирается из справочника "Страны мира".
  • Номер ГТД. Указывается номер грузовой таможенной декларации. Выбирается из справочника "Номера ГТД".
  • Комментарий. Произвольный комментарий к документу.
  • Товары. Табличная часть:
    • Номенклатура. В данное поле вводится элемент справочника номенклатуры.
    • Характеристика. При необходимости в данном поле указывается характеристика номенклатурной позиции.
    • Партия. Указывается партия номенклатурной позиции.
    • Ед. Единица измерения, в которой вводится количество номенклатуры. 
    • Количество. Указывается количество единиц измерения номенклатурной позиции. 
    • Доля стоимости. Используется при разборке для расчета пропорции разделения себестоимости исходного комплекта между комплектующими.
    • Страна происхождения номенклатурной позиции. Выбирается из справочника "Страны мира".

    • Номер ГТД. Указывается номер грузовой таможенной декларации. Выбирается из справочника "Номера ГТД".
