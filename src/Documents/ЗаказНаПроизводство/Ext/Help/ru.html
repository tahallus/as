Документ "Заказ на производство"

Заказ складу на комплектацию товаров или производственному подразделению на производство продукции, выполнение работ, оказание услуг.

Данный документ фактически формирует объемно-календарный план комплектации, производства запасов, выполнения работ, оказания услуг.

В шапке документа указывается структурная единица организации - склад или производственное подразделение, а также дата и время начала выполнения заказа (старт) и время и дата окончания выполнения заказа (финиш).

В табличной части "Продукция" указывается номенклатура, которую необходимо произвести (вид операции Сборка) или разобрать (вид операции Разборка).

В табличной части "Материалы" формируется потребность в сырье, материалах, полуфабрикатах, требующихся для изготовления изделия (вид операции Сборка) или указывается номенклатура - выходной продукт, т.е. результат разборки (вид операции Разборка).

В табличной части "Распределение" выполняется распределение данных из таблицы материалов по производимой продукции в соответствии с выбранными спецификациями. Таблица отображается только при включенном режиме ручного распределения.

В табличной части "Операции" формируется список плановых тех. операций, которые требуется выполнить для производства изделий.

В табличной части "Задействованные ресурсы" указывается перечень ресурсов предприятия, загрузку которых необходимо планировать.

Если используется механизм резервирования запасов, то заказ можно открыть под заказ покупателя, тем самым разместив потребности заказа покупателя по товарам в ожидаемом выпуске по заказу. Необходимые материалы также могут быть зарезервированы на указанном складе под заказ покупателя.

Если используется поэтапное производство с возможностью выполнять этапы разными подразделениями, то для продукции требуется указывать подразделение завершающего этапа - структурную единицу, на которой будет храниться остаток незавершенного производства.

Есть опциональная возможность указывать заказ покупателя и структурную единицу резерва в табличной части (отдельно для каждой строки). Положение изменяется с помощью команды "Шапка / табличная часть" в подменю "Еще" документа.

Использование механизмов резервирования запасов, технологических операций и планирования ресурсов предприятия можно включить в настройках программы.

Документ "Заказ на производство" может быть введен на основании документов:

  • "Заказ покупателя";
  • "Заказ на производство".


Режим "Остатки и резервы":


  • Не использовано - осталось использовать материалов в рамках заказа. (Скрыта по умолчанию. Можно настроить видимость через Еще -> Настройка формы)

  • Зарезервировано (всего) - зарезервировано материалов в рамках заказа на всех складах.

  • Размещено - размещено (заказано) материалов в документах "Заказ на производство" и "Заказ поставщику" по данному заказу.

  • Не обеспечено - не зарезервировано и не заказано материалов по заказу.

  • Свободно (всего) - количество свободных материалов на всех складах.

  • Склад (резерв) - Склад на котором будет произведен резерв при вводе значения в колонку "В резерв". Отображается при переключении расположения склада в документах отгрузки с шапки на табличную часть.

  • Свободно для резерва - свободное количество на складе из колонки "Склад (резерв)". 

  • К поступлению - ожидаемое количество поступления свободных (не привязанной к заказам) материалов.

  • Цветовая индикация колонок:
  • Можно использовать (всего):

    • Зеленый - доступно к отгрузке в полном объеме

    Оранжевый - доступно к отгрузке частично.

    Красный - не доступно  к отгрузке.

  • Обеспечено (всего):

    • Зеленый - обеспеченно полностью.

    Оранжевый - обеспечено частично.

    Красный - не обеспечено.

  • Расшифровки гиперссылок:

  • Зарезервировано (всего) - отображает резервы по всем структурным единицам.

  • Размещено (всего) - отображает размещение номенклатуры в разрезе документов.

  • Свободно (всего) - отображает свободный (не зарезервированный) остаток по всем структурным единицам.

  • К поступлению - отображает ожидаемое к поступлению количество свободной номенклатуры (Заказы поставщикам не привязанные к Заказам покупателя).

  • Дата поступления - ожидаемая дата поступления свободной номенклатуры. Если ожидается поступление по нескольким документам, в расшифровке отражаются соответствующие документы.
