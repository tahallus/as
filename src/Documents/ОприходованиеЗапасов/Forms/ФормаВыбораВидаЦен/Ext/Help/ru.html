Установка цен в документе

Дополнительный диалог "Цены" для заполнения и изменения данных об установке порядка оплаты открывается при нажатии кнопки "Редактировать цены" в форме документа.

При открытии диалога показываются установленные на текущий момент времени значения параметров:

  • Вид цен - выбирается из справочника "Виды цен". Цены позиций номенклатуры, добавляемых в спецификацию документа, будут устанавливаться исходя из заданных для них значений выбранного вида ("Розничные", "Оптовые" и т. д.). В любой момент времени можно выбрать другой вид цен для заполнения цен в спецификации документа. Для того чтобы произошло изменение уже заполненных цен, в документе необходимо установить флажок "Перезаполнить цены".
