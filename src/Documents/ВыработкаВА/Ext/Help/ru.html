Документ "Выработка имущества"

Документ предназначен для фиксирования выработки имущества, используется для расчетов амортизации в случаях применения способа пропорционально объему продукции.

 

Список документов хранится в журнале документов по учету имущества.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, для которой производится фиксация выработки. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Комментарий. Указывается комментарий к документу.

  • Имущество. Табличная часть:

    • Имущество, для которого фиксируется объем выработки. Выбирается из справочника "Имущество".

    • Количество выработки имущества. Указывается в натуральных единицах.
