Документ "Изменение параметров"

Документ предназначен для отражения изменений параметров начисления амортизации имущества.

 

Список документов хранится в журнале документов по учету имущества.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, для которой производится изменение параметров начисления амортизации имущества. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Комментарий. Указывается комментарий к документу.

  • Имущество. Табличная часть:

    • Имущество, для которого изменяются параметры. Выбирается из справочника "Имущество".

    • Новый Срок использования для вычисления амортизации. Заполняется только для имущества со способом начисления "Линейный".

    • Новый Объем продукции для вычисления амортизации. Заполняется только для имущества со способом начисления "Пропорционально объему продукции".

    • Новая Стоимость для начисления амортизации.

    • Стоимость до изменения применяемая для начисления амортизации.

    • С текущего месяца - признак принятия изменения параметров, начиная с текущего месяца.

    • Новый Счет затрат. Счет отнесения затрат на амортизацию имущества.

    • Счет переоценки. Счет отнесения суммы переоценки.

    • Новая Структурная единица, на которую относятся затраты по амортизации имущества. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение" или "Производство".

    • Новое Направление деятельности, на которое относится начисление амортизации. Выбирается из справочника "Направления деятельности".
