Документ "Корректировка долга"

Документ предназначен для проведения взаимозачетов и переуступок долга с контрагентами.

 

В зависимости от установленного вида операции с помощью данного документа могут быть зафиксированы:

  • Взаимозачет;
  • Переуступка долга покупателя;
  • Переуступка долга поставщику;
  • Корректировка долга покупателя;
  • Корректировка долга поставщику.

Операция "Взаимозачет" предназначена для оформления взаимозачета с контрагентом.

Если необходимо оформить операцию переуступки долга покупателя, то устанавливается вид операции "Переуступка долга покупателя".

В случае оформления операции переуступки долга поставщику устанавливается вид операции "Переуступка долга поставщику".

 

Для любого вида операции необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется взаимозачет или переуступка долга. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Комментарий. Указывается комментарий к документу.

 

Для вида операции "Взаимозачет" необходимо заполнить следующие реквизиты:

  • Покупатель, с которого списывается долг перед нами. Выбирается из справочника "Контрагенты".
  • Поставщик, с которого списывается наш долг. Выбирается из справочника "Контрагенты".
  • Расчеты с покупателем. Закладка, содержащая табличную часть:
    • Договор. Указывается договор с покупателем. Выбирается из справочника "Договоры".
    • Документ возникновения долга покупателя.
    • Заказ. Указывается в случае расчетов по определенному заказу. Выбирается документ "Заказ покупателя".
    • Сумма расчетов. Указывается в валюте расчетов по договору с покупателем.
    • Курс валюты расчетов по договору с покупателем. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Кратность курса валюты расчетов по договору с покупателем. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Сумма учета. Указывается в валюте учета. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Расчеты с поставщиком. Закладка, содержащая табличную часть:
    • Договор. Указывается договор с поставщиком. Выбирается из справочника "Договоры".
    • Документ возникновения долга перед поставщиком.
    • Заказ. Указывается в случае расчетов по определенному заказу. Выбирается документ "Заказ поставщику".
    • Сумма расчетов. Указывается в валюте расчетов по договору с поставщиком.
    • Курс валюты расчетов по договору с поставщиком. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Кратность курса валюты расчетов по договору с поставщиком. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Сумма учета. Указывается в валюте учета. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.

 

Закладка "Реквизиты получателя" становится доступной для заполнения для видов операции "Переуступка долга покупателя" и "Переуступка долга поставщику". На ней содержатся следующие реквизиты:

  • Договор. Указывается договор с контрагентом. Выбирается из справочника "Договоры".
  • Документ. Указывается документ расчетов.
  • Заказ. Указывается заказ.
  • Сумма расчетов. Указывается в валюте расчетов по договору с контрагентом.
  • Курс валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Кратность валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
  • Сумма учета. Указывается в валюте учета. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.

 

Для вида операции "Переуступка долга покупателя" необходимо заполнить табличную часть на закладке "Расчеты с покупателем", реквизиты на закладке "Реквизиты получателя" и реквизиты:

  • Покупатель, с которого списывается долг перед нами. Выбирается из справочника "Контрагенты".
  • Покупатель-получатель, на которого списывается долг. Выбирается из справочника "Контрагенты".

 

Для вида операции "Переуступка долга поставщику" необходимо заполнить табличную часть на закладке "Расчеты с поставщиком", реквизиты на закладке "Реквизиты получателя" и реквизиты:

  • Поставщик, с которого списывается наш долг. Выбирается из справочника "Контрагенты".
  • Поставщик-получатель, на которого переводится наш долг. Выбирается из справочника "Контрагенты".

 

Для вида операции "Корректировка долга покупателя" необходимо заполнить табличную часть на закладке "Расчеты с покупателем" и реквизиты:

  • Корреспонденция - счет отнесения корректировки. Если выбран счет с типом "Прочие доходы", то при проведении документа будет зафиксировано увеличение долга покупателя (уменьшение аванса от покупателя). Если выбран счет с типом "Прочие расходы", то при проведении документа будет зафиксировано уменьшение долга покупателя (увеличение аванса от покупателя).

 

Для вида операции "Корректировка долга поставщику" необходимо заполнить табличную часть на закладке "Расчеты с поставщиком" и реквизиты:

  • Корреспонденция - счет отнесения корректировки. Если выбран счет с типом "Прочие доходы", то при проведении документа будет зафиксировано уменьшение долга перед поставщиком (увеличение аванса поставщику). Если выбран счет с типом "Прочие расходы", то при проведении документа будет зафиксировано увеличение долга перед поставщиком (уменьшение аванса поставщику).
