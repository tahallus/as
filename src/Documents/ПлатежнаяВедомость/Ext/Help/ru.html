Документ "Платежная ведомость"

Документ предназначен для формирования печатной формы платежной ведомости по выплате заработной платы сотрудникам организации.

 

В зависимости от установленного вида операции с помощью данного документа может быть сформирована платежная ведомость:

  • По зарплате,
  • По авансам.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой формируется платежная ведомость. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа. Предлагается выбор из списка: аванс или зарплата.
  • Период регистрации платежной ведомости.
  • Подразделение, для которого формируется платежная ведомость.

 

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования валюты документа и валюты расчетов с сотрудником.

  • Сотрудники. Табличная часть:
    • Сотрудник организации, которому выплачивается заработная плата или аванс. Выбирается из справочника "Сотрудники".
    • Начислено, сумма расчетов с сотрудником. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • К выплате, сумма расчетов с сотрудником. Указывается в валюте документа.
