Документ "Табель"

Документ предназначен для ввода информации об использовании рабочего времени.

 

Необходимо заполнить следующие реквизиты:

  • Организация, данные по которой регистрируются документом. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Период регистрации - месяц, на начисление зарплаты за который должны повлиять заносимые данные.

  • Подразделение. В данном реквизите указывается подразделение, в котором числятся сотрудники. Выбирается из справочника "Структурные единицы".

  • Способ ввода данных. Указывается способ заполнения документа: по дням или в целом за период. В первом случае данные хранятся в разрезе дней, во втором случае данные записываются на первое число месяца.

  • Комментарий. Указывается любая дополнительная информация к документу.

  • Отработанное время по дням. В табличной части для сотрудника указывается количество часов по каждому из трех видов времени за каждый по дням месяца. Данная табличная часть доступна для редактирования, если выбран способ ввода данных "По дням".  Реквизиты табличной части:

    • Сотрудник, по которому вводится информация. Выбирается из справочника "Сотрудники".

    • Должность, на которой работает сотрудник. Выбирается из справочника "Должности".

    • Вид 1, Вид 2, Вид 3 - виды учитываемого времени. В один день можно указать информацию по трем видам времени. Выбирается из справочника "Виды рабочего времени".

    • Часы 1, Часы 2, Часы 3 - количество часов за день по первому, второму или третьему виду времени.

  • Отработанное время за период. В табличной части для сотрудника указывается количество часов и дней за месяц по каждому из шести видов времени. Данная табличная часть доступна для редактирования, если выбран способ ввода данных "В целом за период".  Реквизиты табличной части:

    • Сотрудник, по которому вводится информация. Выбирается из справочника "Сотрудники".

    • Должность, на которой работает сотрудник. Выбирается из справочника "Должности".

    • Вид (1) , Вид (2) ... Вид (6) - виды учитываемого времени. В месяц можно указать информацию по шести видам времени. Выбирается из справочника "Виды рабочего времени".

    • Дней (1) , Дней (2) ... Дней (6) - количество дней за месяц по виду времени.

    • Часов (1) , Часов (2) ... Часов (6) - количество часов за месяц по виду времени.
