Документ "Операция по платежным картам"

Документ "Операция по платежным карам" предназначен для учета оплаты банковскими картами.

Получить наглядное представление поможет видеоролик http://www.youtube.com/watch?v=J8nwRVNgugw

В зависимости от установленного вида операции с помощью данного документа может быть зафиксировано поступление или возврат денежных средств.

 

Операция "Оплата картой (от покупателя)" предназначена для оформления поступления денежных средств от покупателя.

Если необходимо оформить операцию возврата оплаты на карту за возвращенные покупателем ранее позиции номенклатуры, то устанавливается вид операции "Возврат оплаты на карту (покупателю)".

 

Для любого вида операции необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Операции. Указывается вид операции документа. Предлагается выбор из списка: "Оплата картой (от покупателя)" или "Возврат оплаты на карту (покупателю)".
  • Способ оплаты - оплата картой в рознице или онлайн на сайте.
  • Основание. Документ-основание платежа.
  • Касса организации, для которой оформляется поступление. Выбирается из справочника "Кассы".
  • Касса ККМ. Заполняется, если возврат оплаты оформляется на основании чека ККМ.
  • Статья движения денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Сумма. Указывается сумма платежа в валюте расчётов по договору эквайринга.
  • Комментарий. Указывается комментарий к документу.
  • Контрагент. Указывается покупатель. Выбирается из справочника "Контрагенты".
  • Налогообложение. Тип налогообложения налогом на добавленную стоимость (НДС).
  • % Комиссии. Процент комиссии банка, которую он берёт за выполнение эквайринговых операций.
  • Комиссия. Сумма комиссии банка в валюте расчётов по договору с эквайрером.
  • Вид карты. Нужно выбрать значение из списка банковских карт, которые перечислены в табличной части в карточке эквайрингового терминала.
  • № карты. Номер банковской карты, которой произведена оплата.
  • № чека ЭТ. Номер чека эквайрингового терминала.
  • Ссылочный номер. Ссылочный номер эквайринговой операции.
  • Расшифровка платежа. Табличная часть:
    • Договор. Указывается договор с контрагентом, по которому поступила оплата. Выбирается из справочника "Договоры".
    • Аванс. Признак аванса платежа.
    • Документ к зачету, по которому осуществляется оплата (возврат оплаты). Для вида операции "Оплата картой (от покупателя)", указывается только для окончательных платежей. Для вида операции "Возврат оплаты на карту (от покупателя)" указывается всегда.
    • Сумма расчетов. Сумма оплаты. Указывается в валюте расчетов по договору с контрагентом.
    • Курс валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Кратность валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Сумма платежа. Сумма оплаты. Указывается в валюте расчётов по договору эквайринга. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Ставка НДС. Выбирается из справочника "Ставки НДС".
    • Сумма НДС. Указывается сумма НДС.
    • Счет на оплату. Указывается в случае поступления (возврата) платежа по определенному счету на оплату.
    • Заказ. Указывается в случае поступления (возврата) платежа по определенному заказу.
    • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Если настроено подключение эквайрингового терминала, то в форме становится доступна кнопка "Оплатить картой". После нажатия на эту кнопку и указания суммы платежа, программа соединяется с эквайринговым терминалом. После выполнения оплаты на терминале, в форме документа будут автоматически заполнены следующие реквизиты: "Вид карты", "№ карты", "№ чека ЭТ" и "Ссылочный номер".

Если настроено подключения фискального регистратора, то на нём можно пробить чек. Для этого нужно нажать кнопку "Напечатать чек".

 

Для вида операции "Возврат оплаты на карту (покупателю)" становятся видимыми следующие реквизиты:

  • Документ оплаты - документ, которым была оформлена оплата банковской картой. Этот реквизит заполняется в обязательном порядке.
  • Флаг "Эквайрер берет комиссию за возврат денежных средств". Устанавливается в соответствии с настройками эквайрингового терминала. Если флаг установлен, то предполагается, что банк берёт комиссию за выполнение эквайринговых операций.
  • Флаг "Возврат комиссии банка при возврате товаров клиентом". Устанавливается в соответствии с настройками эквайрингового терминала. Если флаг установлен, то предполагается, что при возврате Клиентом товара, оплаченного с использованием карты, вознаграждение за осуществление Банком операции по перечислению сумы, уплаченной за возвращенный товар, не взимается.

Удобно оформлять возврат оплаты на карту на основании документа оплаты картой. В этом случае большая часть информации будет заполнена автоматически.

Для оформления возврата оплаты по чеку ККМ или отчёту о розничных продажах, требуется оформить новый документ "Операция по платежным картам", выбрать операцию "Возврат оплаты на карту (покупателю)". Станет доступен реквизит "Документ оплаты", в котором нужно выбрать документ-основание.

Документ "Поступление в кассу" может быть введен на основании документов:

  • "Акт выполненных работ",
  • "Расходная накладная",
  • "Приходная накладная" (операция "Возврат от покупателя"),
  • "Продажа имущества",
  • "Счет на оплату",
  • "Перемещение денег (план)",
  • "Поступление денег (план)",
  • "Заказ покупателя",
  • "Прием и передача в ремонт",
  • "Отчет о розничных продажах".
