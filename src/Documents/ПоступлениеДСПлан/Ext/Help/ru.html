Документ "Поступление денег (план)"

Документ "Поступление денег (план)" предназначен для планирования поступления денежных средств.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется планирование. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Валюта. Указывается валюта планирования. Выбирается из справочника "Валюты".
  • Сумма. Указывается планируемая сумма платежа в валюте планирования.
  • Статья движения денежных средств, по которой планируется поступления денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Способ оплаты. Указывается одним из вариантов: безналичные, наличные или не определен.
  • Контрагент, от которого планируется поступление денег. Выбирается из справочника "Контрагенты".
  • Договор. Указывается договор с контрагентом, по которому планируется поступление оплаты. Выбирается из справочника "Договоры".
  • Банковский счет организации, для которой планируется поступление. Становится доступным для указания при выбранном способе оплаты - безналичные. Выбирается из справочника "Банковские счета".
  • Касса организации, для которой планируется поступление. Становится доступным для указания при выбранном способе оплаты - наличные. Выбирается из справочника "Кассы".
  • Документ - основание планирования поступления.
  • Комментарий. Указывается комментарий к документу.
