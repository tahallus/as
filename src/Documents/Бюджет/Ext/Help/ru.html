Документ "Бюджет"

Документ предназначен для хранения информации о бюджете компании на определенный период планирования.

Планирование происходит по суммовым показателям.

 

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, для которой формируется прогнозный баланс. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Период планирования. В данном реквизите указывается период планирования, который определяет периодичность, начало и окончание прогнозного баланса. Выбирается из справочника "Периоды планирования".

  • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.

  • Начальные остатки. Табличная часть:

    • Счет остатков. Выбирается из Плана счетов "Управленческий".

    • Сумма остатка.

    • Комментарий к операции.

  • Прямые затраты. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Счет (статья затрат). Выбирается из Плана счетов "Управленческий".

    • Источник затрат (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма затрат.

    • Комментарий к операции.

  • Косвенные затраты. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Счет (статья затрат). Выбирается из Плана счетов "Управленческий".

    • Источник затрат (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма затрат.

    • Комментарий к операции.

  • Поступления денежных средств. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Банк, касса (счет учета денежных средств). Выбирается из Плана счетов "Управленческий".

    • Источник поступления (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма.

    • Статья (статья движения денежных средств). Выбирается из Справочника "Статьи движения денежных средств".

    • Комментарий к операции.

  • Выбытия денежных средств. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Банк, касса (счет учета денежных средств). Выбирается из Плана счетов "Управленческий".

    • Направление выбытия (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма.

    • Статья (статья движения денежных средств). Выбирается из Справочника "Статьи движения денежных средств".

    • Комментарий к операции.

  • Доходы. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Статья доходов (счет учета доходов). Выбирается из Плана счетов "Управленческий".

    • Источник (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма.

    • Направление деятельности. Выбирается из Справочника "Направления деятельности".

    • Структурная единица (подразделение). Выбирается из Справочника "Организационно-структурные единицы".

    • Заказ (проект) - заказ покупателя.

    • Комментарий к операции.

  • Расходы. Табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Статья расходов (счет учета расходов). Выбирается из Плана счетов "Управленческий".

    • Источник (корр. счет). Выбирается из Плана счетов "Управленческий".

    • Сумма.

    • Направление деятельности. Выбирается из Справочника "Направления деятельности".

    • Структурная единица (подразделение). Выбирается из Справочника "Организационно-структурные единицы".

    • Заказ (проект) - заказ покупателя.

    • Комментарий к операции.

  • Операции. Прочие бюджетные операции, табличная часть:

    • Дата планирования - начало планового периода. При вводе данных реквизит автоматически выравнивается на начало планового периода, определенного периодом планирования.

    • Счет Дт проводки. Выбирается из Плана счетов "Управленческий".

    • Счет Кт проводки. Выбирается из Плана счетов "Управленческий".

    • Сумма проводки.

    • Комментарий к операции.
