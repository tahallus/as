Документ "Счет на оплату"

Документ предназначен для формирования печатной формы счета на оплату покупателю.

 

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования валюты данного документа.

В подвале формы можно указать ручную скидку на документ, процентом или суммой.

Для формирования печатной формы нужно заполнить следующие реквизиты:

  • Организация, от которой выписывается счет на оплату. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Контрагент.  В данном реквизите указывается информация о покупателе. Выбирается из справочника "Контрагенты".
  • Договор. В данном реквизите указывается договор контрагента, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".

Если выбрано "Запланировать оплату":

  • Способ оплаты. Указывается один из вариантов: безналичные, наличные или не определен.
  • Банковский счет организации, для оплаты. Становится доступным для указания при выбранном способе оплаты безналичные. Выбирается из справочника "Банковские счета".
  • Касса организации, в которую планируется приход. Становится доступным для указания при выбранном способе оплаты наличные. Выбирается из справочника "Кассы".

Если документ вводится на основании другого:

  • Основание - документ, на основании которого выписывается счет на оплату.

  • Запасы. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, требование оплаты которой отражается данным документом. Выбирается из справочника "Номенклатура" с типом "Запас", "Услуга" или "Работа".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит обязателен для заполнения, если вид операции документа: "Возврат комитенту", "Возврат из переработки" или "Возврат с ответственного хранения". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество номенклатурной позиции, требующей оплаты.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Цена - цена реализации номенклатурной позиции.

    • % Скидки - процент ручной скидки (наценки).

    • % Авт. - процент автоматической скидки (наценки). Процент автоматической скидки редактируется в справочнике "Виды автоматических скидок и наценок". Расчёт автоматических скидок в документе осуществляется при записи документа и при нажатии кнопки "% Авт." над табличной частью.

      Сумма - стоимость реализуемой номенклатуры по данной строке, с налогами согласно флажкам.

    • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - поле, автоматически заполняемое суммарной стоимостью запасов.

    • Содержание - представление запаса в печатной форме. Чтобы реквизит стал видимым, необходимо выполнить "Все действия" - "Изменить форму" и установить флаг напротив реквизита.

  • Доставка. Группа реквизитов:

    • Услуга - номенклатура с типом "Услуга", использующаяся для отражения доставки в учете. Обязательный реквизит при использовании способа доставки отличного от "Самовывоз". Выбирается из справочника "Номенклатура".

    • Стоимость - стоимость доставки заказа.

    • % НДС, сумма НДС - данные НДС доставки.

    • Объем - общий объем заказа. Рассчитывается автоматически по объему элементов из табличной части "Запасы". Может быть отредактирован вручную.

    • Вес - вес заказа. Рассчитывается автоматически по весу элементов из табличной части "Запасы". Может быть отредактирован вручную.

  • Дополнительная информация. Группа реквизитов:

    • Подразделение - информация о подразделении-исполнителе. Выбирается из справочника "Структурные единицы" с типом "Административно-управленческое подразделение".

    • Ответственный - информация о сотруднике, ответственном за ввод текущего документа. Выбирается из справочника "Сотрудники". Реквизит не обязателен для заполнения.

    • Комментарий. Указывается комментарий к документу.

    • % предоплаты. Процент предоплаты по счету.

    • Сумма предоплаты. Сумма предоплаты по счету.

    • НДС предоплаты. Сумма НДС предоплаты по счету.

    • Дата оплаты. Дата полной оплаты по счету.

    • Дополнительные условия - текст дополнительных условий поставки в печатной форме счета, который можно выбрать из существующих шаблонов.


Минимальные цены


При включении опции "Минимальные цены продажи" и настройки минимальных цен (Раздел "Продажи" -> "Еще больше возможностей") в табличной части цвет текста поля цена будет изменен на красный, если значение цены станет меньше чем минимальная.
Минимальные цены будут подбираться из общего вида минимальных цен.
