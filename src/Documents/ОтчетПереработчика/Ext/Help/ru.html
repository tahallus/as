Документ "Отчет переработчика"

Документ предназначен для хранения информации об отражении в учете факта переработки сырья и материалов, которые были переданы стороннему контрагенту.

 

Необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от имени которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Основание. В данном реквизите указывается "Заказ поставщику", на основании которого сформирован данный документ. Реквизит необязателен для заполнения.

  • Заказ покупателя. В данном реквизите указывается источник потребности. Выбирается документ "Заказ покупателя". С помощью пункта меню Шапка / табличная часть можно переключиться в режим размещения заказа покупателя в табличных частях. Реквизит необязателен для заполнения.

  • Подразделение. В данном реквизите указывается структурная единица, на которую приходуется результат переработки. Выбирается из справочника "Структурные единицы" с типом "Склад" или "Производство".

  • Ячейка. В данном реквизите указывается ячейка структурной единицы (стеллаж, участок), на которую приходуется результат переработки. Выбирается из справочника "Ячейки". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по ячеистым складам.

  • Контрагент. В данном реквизите указывается переработчик. Выбирается из справочника "Контрагенты".

  • Договор. В данном реквизите указывается договор контрагента, в рамках которого оформляется документ. Выбирается из справочника "Договоры контрагентов".

 

На форме документа расположена гиперссылка "Редактировать цены и валюту", по которой вызывается диалог редактирования видов цен и валюты данного документа.

  • Продукция. Группа реквизитов:

    • Номенклатура - позиция номенклатуры, представляющая собой результат переработки сырья и материалов, переданных переработчику. Выбирается из справочника "Номенклатура" с типом "Запас".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - при необходимости в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество изготовленной номенклатурной позиции переработчиком.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Спецификация - спецификация, содержащая сведения о технологии производимой продукции. Выбирается из справочника "Спецификации". Реквизит необязателен для заполнения.

    • Заказ покупателя - заказ покупателя, под который будет выполнено резервирование готовой продукции. Доступен только после включения опции Резервирование запасов в настройках программы.

  • Услуги. Группа реквизитов:

    • Номенклатура - услуга по переработке. Выбирается из справочника "Номенклатура" с типом "Услуга".

    • Сумма - стоимость услуги.

    • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

    • Сумма НДС - сумма налога на добавленную стоимость.

    • Всего - поле, автоматически заполняемое суммарной стоимостью услуг, согласно настройкам цен и валют. Настройки устанавливаются в диалоге, вызываемом по кнопке "Редактировать цены и валюту".

    • С помощью пункта меню Шапка / табличная часть можно переключиться в режим ввода нескольких услуг. При этом вместо реквизитов на закладке отобразится табличная часть с соответствующими колонками.

  • Запасы. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, использованная при переработке сырья и материалов переработчиком. Выбирается из справочника "Номенклатура" с типом "Запас".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - при необходимости в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество использованной номенклатурной позиции переработчиком.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

    • Спецификация - спецификация, содержащая сведения о технологии производимых полуфабрикатов. Выбирается из справочника "Спецификации". Реквизит необязателен для заполнения.

    • Цена - цена использованной номенклатурной позиции.

    • Сумма - стоимость номенклатуры по данной строке.

    • Ставка НДС - ставка налога на добавленную стоимость. Выбирается из справочника "Ставки НДС".

    • Сумма НДС - сумма налога на добавленную стоимость по данной строке.

    • Всего - поле, автоматически заполняемое суммарной стоимостью запасов, которые были переданы переработчику, согласно настройкам цен и валют. Настройки устанавливаются в диалоге, вызываемом по гиперссылке "Редактировать цены и валюту".

    • Заказ покупателя - заказ покупателя, из резерва которого будут списаны материалы. Доступен только после включения опции Резервирование запасов в настройках программы.

  • Отходы. Данная табличная часть может быть заполнена с помощью подбора (кнопка "Подбор" командной панели табличной части) либо методом ручного ввода. Реквизиты табличной части:

    • Номенклатура - позиция номенклатуры, представляющая собой отход, возникший в результате изготовления продукции и полуфабрикатов переработчиком. Выбирается из справочника "Номенклатура" с типом "Запас".

    • Характеристика - при необходимости в данном поле указывается характеристика номенклатуры. Выбирается из справочника "Характеристики номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по характеристикам номенклатуры.

    • Партия - при необходимости в данном поле указывается партия номенклатуры. Выбирается из справочника "Партии номенклатуры". Реквизит отображается на форме, если в настройках параметров учета включено ведение партионного учета.

    • Количество - количество отходов.

    • Ед. - единица измерения номенклатурной позиции. Выбирается из справочника "Классификатор единиц измерения" или "Единицы измерения".

  • Предоплата. Данная табличная часть заполняется сведениями по зачету авансов переработчику. На форме документа расположена гиперссылка "Редактировать зачет предоплаты", по которой вызывается диалог редактирования табличной части "Зачет предоплаты". Реквизиты табличной части:

    • Документ - документ поступления аванса от покупателя: "Расход со счета", "Расход из кассы".

    • Сумма расчетов - сумма в валюте договора.

    • Курс - курс пересчета в валюту документа.

    • Кратность - кратность пересчета в валюту документа.

    • Сумма платежа - указывается в валюте документа.

  • Дополнительная информация. Группа реквизитов:

    • Комментарий - поле ввода, предназначенное для указания любой дополнительной информации к документу.

 

Документ "Отчет переработчика" может быть введен на основании документов:

  • "Заказ поставщику".
