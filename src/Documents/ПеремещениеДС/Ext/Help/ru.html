Документ "Перемещение денег"

Документ "Перемещение денег" предназначен для осуществления перемещения денежных средств.

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется документ. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Валюта. Указывается валюта денежных средств. Выбирается из справочника "Валюты".
  • Сумма. Указывается сумма перемещения в валюте денежных средств.
  • Статья движения денежных средств, по которой осуществляется перемещение. Выбирается из справочника "Статьи движения денежных средств".
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)".
  • Источник денежных средств. Группа реквизитов:
    • Тип денежных средств источника. Указывается один из вариантов: безналичные или наличные.
    • Банковский счет источника организации, для которого осуществляется перемещение. Становится доступным для указания при выбранном типе денежных средств - безналичные. Выбирается из справочника "Банковские счета".
    • Касса источника организации, для которой осуществляется перемещение. Становится доступным для указания при типе денежных средств - наличные. Выбирается из справочника "Кассы".
  • Получатель денежных средств. Группа реквизитов:
    • Тип денежных средств получателя. Указывается один из вариантов: безналичные или наличные.
    • Банковский счет получателя организации, на который осуществляется перемещение. Становится доступным для указания при типе денежных средств - безналичные. Выбирается из справочника "Банковские счета".
    • Касса получателя организации, в которую осуществляется перемещение. Становится доступным для указания при типе денежных средств - наличные. Выбирается из справочника "Кассы".
  • Комментарий. Указывается комментарий к документу.
