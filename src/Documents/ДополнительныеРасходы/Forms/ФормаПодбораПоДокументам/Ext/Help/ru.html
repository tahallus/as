Форма "Подбор по документам"

Форма предназначена для подбора запасов в соответствующую табличную часть документа "Дополнительные расходы" из одного или нескольких документов вида "Приходная накладная" или "Авансовый отчет".

Чтобы заполнить табличную часть документа, необходимо:

  • в табличную часть "Отобранные документы" занести документы, из которых будит выбранная номенклатура для распределения суммы расходов;
  • если требуется выбрать к распределению дополнительных расходов не весь перечень номенклатуры из указанных документов, то в табличной части "Номенклатура отбора" требуется указать перечень номенклатуры, которая доступна для переноса, и включить опцию "Использовать отбор по указанной номенклатуре";
  • в табличную часть "Отобранная номенклатура для документа" будет добавлена выбранная номенклатура;
  • по кнопке "Ок" информация из табличной части "Отобранная номенклатура для документа" будет перенесена в табличную часть документа. 

 

При включении опции "Отображать запасы выбранные из текущего документа" в табличной части "Отобранная номенклатура для документа" будут отображаться запасы, выбранные только из текущего документа табличного поля "Отобранные документы".

Если опция "Добавлять новые позиции в подвал таблицы" включена, новые строки при заполнении из выбранных документов будут добавляться в подвал табличной части "Отобранная номенклатура для документа" даже если существует аналоги. Если опцию выключить, то данные из новой строки будут суммироваться в строку с аналогичными параметрами. Поиск схожих строк выполняется по следующим полям: документ поступления, номенклатура, характеристика, партия, единица измерения, ставка НДС, цена.
