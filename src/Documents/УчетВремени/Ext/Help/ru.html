Документ "Учет времени"

Документ предназначен для регистрации времени, затраченного на выполнение задания.

 

Документ "Учет времени" может быть введен на основании документа "Задание на работу".

 

В документе необходимо заполнить следующие реквизиты:

  • Организация, в которой регистрируется отработанное время. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Подразделение, в котором регистрируется отработанное время. Выбирается из справочника "Структурные единицы".
  • Сотрудник, по которому регистрируется выполнение задания. Выбирается из справочника "Сотрудники".
  • Неделя с ... по ... - неделя, за которую ведется регистрация отработанного времени.
  • Комментарий. Указывается любая дополнительная информация к документу.
  • Операции. В табличной части указывается отработанное время с возможностью детализации до вида работ. Для упрощения заполнения табличной части предусмотрен механизм автоматического заполнения по запланированным заданиям на работу. Реквизиты табличной части:
    • Заказчик, для которого выполнена работа. Выбирается из справочника "Контрагенты", "Договоры контрагентов", "Заказы покупателей".
    • Вид работ, по которому выполнена работа. Выбирается из справочника "Номенклатура".
    • Работа (услуга), для изготовления которой выполнена работа. Выбирается из справочника "Номенклатура".
    • Характеристика работы (услуги), для изготовления которой выполнена работа. Выбирается из справочника "Характеристики номенклатуры".
    • Всего - количество часов за неделю, затраченное на выполнение задания.
    • Вид цен, по которому определяется стоимость работ для заказчика. Выбирается из справочника "Виды цен".
    • Расценка - стоимость часа работы по заданию.
    • Сумма - стоимость всех работ в строке.
    • Длительность - длительность выполнения задания в часах.
    • С ... по - время начала выполнения задания и время окончания.
    • Комментарий. Указывается любая дополнительная информация к строке.
