Форма "Калькуляция"

Форма предназначена для корректировки состава спецификаций и расчета предварительной себестоимости заказа

Работа с калькуляцией возможна в двух режимах: с отображением данных по себестоимости и прибыли или с отображением только данных по количеству и сумме продажи. При включенном отображении данных по себестоимости, пользователь может выбирать по какой цене рассчитывать себестоимость заказа: по последней цене закупки или по одному из заведенных в базе видов цен.
