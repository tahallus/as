Документ "Доверенность"

Документ предназначен для оформления доверенностей на получение товаров.

В документ вносятся данные о доверенном лице путем выбора элемента справочника "Физические лица".

 

В документе необходимо заполнить следующие реквизиты:

  • Организация. В данном реквизите указывается собственная организация, от которой выписывается доверенность. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.

  • Основание. Указывается документ-основание выписки доверенности. Выбирается документ "Заказ поставщику".

  • Доверенное лицо, на которое выписывается доверенность. Выбирается из справочника "Физические лица".

  • Дата действия доверенности.

  • Банковский счет организации. Отображается в печатной форме документа. Выбирается из справочника "Банковские счета".

  • Контрагент, для которого выписывается доверенность. Выбирается из справочника "Контрагенты".

  • Договор контрагента. Выбирается из справочника "Договоры контрагентов".

  • На получение от. Указывается полное наименование контрагента. Реквизит отображается в печатной форме документа.

  • По документу. Указывается полное наименование документа-основания. Реквизит отображается в печатной форме документа.

  • Комментарий. Указывается комментарий к документу.

  • Запасы. Табличная часть:

    • Наименование товара, получаемого по доверенности.

    • Ед. Указывается единица измерения товара.

    • Количество единиц к получению по доверенности.
