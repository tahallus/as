Документ "Установка цен"

Документ предназначен для расчета, изменения цен номенклатуры. 

Документ "Установка цен" может быть введен на основании документа "Приходная накладная":

  • "Приходная накладная"

    • Настройки документа:

  • "Показывать характеристики" - настройка доступна при включенной опции "Характеристики". Позволяет указывать цены на конкретные характеристики по номенклатуре.

  • "Действующую цену" - при включенной настройке в таблице номенклатуры будут отображены действующие цены.

  • "Отклонение цен" - при включенной настройке в таблице номенклатуры будут отображены изменения цены.

  • "Единицы измерения" - при включенной настройке в таблице номенклатуры будут отображены единицы измерения.

  • "Зависимые цены" - при включенной настройке в таблице номенклатуры будут добавлены колонки всех зависимых от расчетного вида цен, данные колонки будут доступны только для просмотра.

  • "Записывать новые цены поверх установленных" - если установлена данная настройка, то при проведении документа произойдет обновление данных.

    • Пример. Для номенклатуры заведена цена на 01.06 в размере 10 рублей. В документе для этой же номенклатуры указана цена 15 и введена дата 01.06 аналогичного года. 
    При выключенной опции, после проведения, цена будет равна 10 рублям, а при включенной 15 рублей.

  • "Устанавливать характеристикам без цен" - если установлена данная настройка, то при проведении характеристикам, у которых не указаны цены будет назначена цена из номенклатуры.


    • На закладке "Товары" определены две группы команд. Подобрать номенклатуру: 

    • по приходным накладным – добавить номенклатуру из приходной накладной 
    • по номенклатурной группе – добавить всю номенклатуру, которая относится к определенной номенклатурной группе 
    • по ценовой группе - добавить всю номенклатуру, которая относится к определенной ценовой группе 
    • по виду цен – команда позволяет добавить номенклатуру, для которой не заполнены цены, или у которой есть записи по виду цен (колонкам прайс-листа)
    • из списка номенклатуры – добавить произвольную номенклатуру из общего списка номенклатуры (реализовано две возможности: добавление по элементам, добавление по группам) 

     

    Команды группы Действия: 

    • Изменить на сумму 
    • Изменить на процент 
    • Округление 
    • Очистить 
    • Изменить по формуле
