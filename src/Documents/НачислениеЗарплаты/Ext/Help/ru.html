Документ "Начисление зарплаты"

Документ предназначен для начисления зарплаты сотрудникам и отнесения ее на затраты.

Документ может быть заполнен вручную, а могут быть использованы механизмы Заполнения и Расчета документа, которые находятся в командной панели.

В документе необходимо заполнить следующие реквизиты:

  • Организация, в которой происходит начисление зарплаты. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Комментарий. Указывается любая дополнительная информация к документу.
  • Период регистрации. Указывается месяц начисления зарплаты.
  • Подразделение. В данном поле указывается подразделение, по которому рассчитывается зарплата. Выбирается из справочника "Структурные единицы".
  • Валюта документа. Валюта, в которой происходит начисление. Документ заполняется только теми плановыми начислениями, у которых установлена валюта, равная валюте документа. Т.е. для каждой валюты начисления необходимо создавать отдельный документ. Выбирается из справочника "Валюты". Реквизит отображается на форме, если в настройках параметров учета включено ведение валютного учета.

  • Начисления и удержания. Данная табличная часть содержит информацию о начисленной сотрудникам заработной плате и способе отнесения ее на затраты. Реквизиты табличной части:

    • Сотрудник, которому начисляется заработная плата. Выбирается из справочника "Сотрудники".
    • Должность - в данное поле подставляется должность, на которой работает сотрудник в указанный в строке период. Выбирается из справочника "Должности".
    • Вид начисления (удержания) - начисление или удержание. Выбирается из справочника "Виды начислений и удержаний".

    • Дата начала, Дата окончания - период, за который начисляется зарплата сотруднику с выбранными параметрами.

    • Отработано дней - в данном поле указывается количество дней, которое работник отработал на указанной должности с указанными параметрами начисления зарплаты (автоматически заполняется по табелю).

    • Отработано часов - в данном поле указывается количество часов, которое работник отработал на указанной должности с указанными параметрами начисления зарплаты (автоматически заполняется по табелю).

    • Размер планового начисления или удержания в валюте документа. 

    • Сумма - рассчитанная сумма начисления или удержания.

    • Счет затрат - счет затрат, на который будет отнесена сумма начисления или удержания.

    • Направление деятельности, на которое будет отнесена сумма начисления или удержания. Выбирается из справочника "Направления деятельности".

    • Заказ покупателя, на который будет отнесена сумма начисления или удержания. Выбирается из списка документов "Заказы покупателей".

    • К - признак ручной корректировки строки. Установленный флажок говорит о том, что эту строку необходимо исключить из расчета. Сумма перезаполнена не будет.

    • Показатель 1, Показатель 2, Показатель 3 - параметры расчета, значения которых требуются для расчета по формуле вида начисления.

    • Значение 1, Значение 2, Значение 3 - значения показателей.  Рассчитываются при автозаполнении, доступны для редактирования.

  • Налоги на доходы. В табличную часть вручную вносятся данные о начисленных налогах на доходы. Табличная часть отображается на форме, если в настройках параметров учета включено ведение учета налога на доходы.

    • Сотрудник, которому начислен налог. Выбирается из справочника "Сотрудники".
    • Вид начисления (удержания) - налог. Выбирается из справочника "Виды начислений и удержаний".
    • Сумма - вносимая сумма налога.
