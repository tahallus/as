Документ "Расход со счета"

Документ "Расход со счета" предназначен для учета расхода безналичных денежных средств.

 

В зависимости от установленного вида операции с помощью данного документа может быть зафиксирован расход наличных денежных средств по различным операциям:

  • Поставщику,
  • Покупателю,
  • Подотчетнику,
  • Зарплата по ведомости,
  • Зарплата сотруднику,  
  • Налоги,
  • Личные средства,
  • Прочий расход,
  • На расходы,
  • Отчет эквайрера.


Операция "Поставщику" предназначена для оформления оплаты поставщику безналичными денежными средствами.

Если необходимо оформить операцию оплаты покупателю за возвращенные им ранее позиции номенклатуры, то устанавливается вид операции "Покупателю".

В случае оформления операции выдачи денежных средств подотчетнику устанавливается вид операции "Подотчетнику".

Для оформления расходования денежных средств на выплату заработной платы по ведомости устанавливается вид операции "Зарплата по ведомости".

Для оформления расходования денежных средств на выплату заработной платы конкретному сотруднику устанавливается вид операции "Зарплата сотруднику".

В случае оплаты налогов устанавливается вид операции "Налоги".

Для оформления вывода личных средств со счета предпринимателя используется вид операции "Личные средства".

Если нужно отразить списание средств от эквайрера за возврат оплаты на платежные карты, то выберите операцию "Отчет эквайрера". Операция подходит в том случае, если сумма возвратов на карты превышает сумму поступления с карт.

В остальных случаях выбирается операция "Прочее".

 

Для любого вида операции необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется расход со счета. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа. Предлагается выбор из списка: поставщику, покупателю, подотчетнику, зарплата по ведомости, зарплата сотруднику, налоги или прочее.
  • Банковский счет организации, для которой оформляется расход. Выбирается из справочника "Банковские счета".
  • Статья движения денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Сумма. Указывается сумма платежа в валюте расчетного счета.
  • Основание. Документ-основание платежа.
  • Назначение платежа.
  • Комментарий. Указывается комментарий к документу.

 

Для вида операции "Поставщику" или "Покупателю" становятся видимыми следующие реквизиты:

  • Контрагент. В зависимости от вида операции указывается покупатель или поставщик. Выбирается из справочника "Контрагенты".
  • Счет контрагента, выбирается из справочника "Банковские счета".
  • Налогообложение. Тип налогообложения продажи товаров и услуг налогом на добавленную стоимость.
  • Расшифровка платежа. Табличная часть или группа реквизитов, в зависимости от установленного представления ("Строкой" / "Списком"):
    • Договор. Указывается договор с контрагентом, по которому производится оплата. Выбирается из справочника "Договоры".
    • Аванс. Признак аванса платежа.
    • Документ отгрузки (Документ расчетов), по которому осуществляется оплата. Для вида операции "Поставщику" указывается только для платежей, которые не являются авансами. Для вида операции "Покупателю" указывается всегда.
    • Сумма расчетов. Сумма оплаты. Указывается в валюте расчетов по договору с контрагентом.
    • Курс валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Кратность валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Сумма платежа. Сумма оплаты. Указывается в валюте расчетного счета. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Ставка НДС. Выбирается из справочника "Ставки НДС".
    • Сумма НДС. Указывается сумма НДС.
    • Оплачиваемый счет. Указывается в случае оплаты по определенному счету. Выбирается из списка документов "Счет на оплату поставщика".
    • Оплачиваемый заказ. Указывается в случае осуществления платежа по определенному заказу.
    • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Для вида операции "Подотчетнику" становятся видимыми следующие реквизиты:

  • Подотчетник, которому выдаются денежные средства. Выбирается из справочника "Сотрудники".
  • Документ перерасхода. Заполняется в случае перерасхода подотчетником денежных средств по авансовому отчету. Выбирается из списка документов "Авансовые отчеты".
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или "Расход денег (план)".

 

Для вида операции "Зарплата по ведомости" становятся видимыми следующие реквизиты:

  • Сотрудник, которому выдаются денежные средства. Выбирается из справочника "Сотрудники".
  • Направление деятельности, отнесения расходов по кассовому методу учета доходов и расходов. Выбирается из справочника "Направления деятельности". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов.
  • Выплата заработной платы. Табличная часть:
    • Ведомость. Платежная ведомость, по которой выплачиваются денежные средства. Выбирается из списка документов "Платежные ведомости".
    • Сумма платежа. Сумма выплаты заработной платы по ведомости. Указывается в валюте кассы.
    • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или "Заявка на расход денег (план)".

 

Для вида операции "Зарплата сотруднику" становятся видимыми следующие реквизиты:

  • Период регистрации платежа.
  • Подразделение, на которое относится платеж. Выбирается из справочника "Структурные единицы". Указывается только если корреспонденция имеет тип - расходы.
  • Направление деятельности, отнесения расходов по кассовому методу учета доходов и расходов. Выбирается из справочника "Направления деятельности". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или "Заявка на расход денег (план)".

 

Для вида операции "Налоги" становятся видимыми следующие реквизиты:

  • Вид налога. Выбирается из справочника "Виды налогов".
  • Направление деятельности, отнесения расходов по кассовому методу учета доходов и расходов. Выбирается из справочника "Направления деятельности". Реквизит отображается на форме, если в настройках параметров учета включено использование кассового метода учета доходов и расходов.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или "Расход денег (план)".

 

Для вида операции "Отчет эквайрера" становятся видимыми следующие реквизиты:

  • Банк-эквайрер. Указывается банк, который вернул деньги на платежные карты покупателей. Выбирается из справочника "Контрагенты".
  • Терминал. Эквайринговый терминал, по которому был проведен возврат оплаты на платежную карту. Выбирается из справочника "Эквайринговые терминалы".
  • Сумма. Сумма текущего платежа.
  • Комиссия. Комиссия банка за операции эквайринга.
  • Подразделение затрат. Подразделение, на которое будет отнесена комиссия банку.
  • Направление деятельности затрат. Направление деятельности, на которое будет отнесена комиссия банку.
  • Расшифровка платежа. Табличная часть:
    • Документ. Документ возврата оплаты на карту. 
    • Дата платежа. Дата, когда был возврат оплаты на карту или оплата картой.
    • Оплата (сумма). Сумма оплаты, которая была списана с карты покупателя. 
    • Возврат (сумма). Сумма возврата на карту покупателя. 
    • Комиссия за оплату и комиссия за возврат. Сумма комиссии, которую банк взимает / возвращает за конкретную операцию оплаты / возврата.
    • Вид карты. Вид карты, например, Visa. 
    • Номер карты. Номер карты покупателя.
    • Заказ. Заказ покупателя, который был оплачен картой (по которому был возврат).

Табличная часть "Расшифровка платежа" может быть заполнена автоматически с помощью кнопок "Заполнить остатками" и "Заполнить за период".

По кнопке "Заполнить остатками" табличная часть будет заполнена всеми документами, по которым ранее не были оформлены отчёты эквайера документами "Поступление на счет" и "Расход со счета".

Кнопка "Заполнить за период" работает аналогично, но дополнительно позволяет указать фильтр по датам документов оплаты картами.


Для вида операции "Прочий расход" становятся видимыми следующие реквизиты:

  • Корреспонденция - счет отнесения платежа.
  • Направление деятельности, на которое относится платеж. Выбирается из справочника "Направления деятельности". Указывается только если корреспонденция имеет тип - расходы.
  • Подразделение, на которое относится платеж. Выбирается из справочника "Структурные единицы". Указывается только если корреспонденция имеет тип - расходы.
  • Заказ покупателя, на который относится платеж. Выбирается документ "Заказ покупателя". Указывается только если корреспонденция имеет тип - расходы.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или "Расход денег (план)".

 

Операция "На расходы" аналогичная операции "Прочий расход".

Но поле "Корреспонденция" называется "Статья расходов".

Статья расходов выбирается из счетов управленческого плана счетов с типом счёта "Расходы (распределяемые на финансовый результат)" или "Прочие расходы (распределяемые на финансовый результат)".

С помощью этой операции рекомендуется оформлять прочие расходы и расходы, которые нельзя отнести на конкретную продукцию, а нужно относить сразу на финансовый результат.

Для получения более детальной аналитики в отчетах рекомендуется для каждой статьи или группы статей расходов создать свой счет управленческого плана счетов.



Документ "Расход со счета" может быть введен на основании документов:

  • "Платежное поручение",
  • "Приходная накладная",
  • "Дополнительные расходы",
  • "Отчет комиссионера",
  • "Отчет комитенту",
  • "Заказ поставщику",
  • "Счет на оплату (полученный)",
  • "Платежная ведомость",
  • "Начисление налогов",
  • "Перемещение денег (план)",
  • "Расход денег (план)".
