Документ "Поступление в кассу"

Документ "Поступление в кассу" предназначен для учета поступлений наличных денежных средств.

 

В зависимости от установленного вида операции с помощью данного документа может быть зафиксировано поступление наличных денежных средств по различным операциям:

  • От покупателя;
  • От поставщика;
  • От подотчетника; 
  • Розничная выручка;
  • Розничная выручка (суммовой учет);
  • Покупка валюты;
  • Личные средства; 
  • Прочее.

Операция "От покупателя" предназначена для оформления поступления наличных денежных средств от покупателя.

Если необходимо оформить операцию получения денег от поставщика за возвращенные ему ранее позиции номенклатуры, то устанавливается вид операции "От поставщика".

В случае оформления операции по возврату неизрасходованных денежных средств подотчетником устанавливается вид операции "От подотчетника".

Если необходимо оформить операцию оприходования розничной выручки, то устанавливается вид операции "Розничная выручка" или "Розничная выручка (суммовой учет)".

При приобретении иностранной валюты устанавливается вид операции "Покупка валюты".

Для оформления взноса личных средств предпринимателя в кассу используется вид операции "Личные средства".

В остальных случаях выбирается операция "Прочее".

 

Для любого вида операции необходимо заполнить следующие реквизиты:

  • Организация, для которой оформляется поступление в кассу. Выбирается из справочника "Организации". Реквизит отображается на форме, если в настройках параметров учета включено ведение учета по нескольким организациям.
  • Вид операции. Указывается вид операции документа. Предлагается выбор из списка: от покупателя, от поставщика, от подотчетника, розничная выручка, розничная выручка (суммовой учет), покупка валюты или прочее.
  • Основание. Документ-основание платежа.
  • Касса организации, для которой оформляется поступление. Выбирается из справочника "Кассы".
  • Статья движения денежных средств. Выбирается из справочника "Статьи движения денежных средств".
  • Сумма. Указывается сумма платежа в валюте кассы.
  • Валюта кассы. Указывается валюта кассы. Выбирается из справочника "Валюты".
  • Принято от. Отображается в соответствующем поле печатной формы документа.
  • Основание платежа. Отображается в соответствующем поле печатной формы документа.
  • Приложение. Отображается в соответствующем поле печатной формы документа.
  • Комментарий. Указывается комментарий к документу.

 

Для вида операции "От покупателя" или "От поставщика" становятся видимыми следующие реквизиты:

  • Контрагент. В зависимости от вида операции указывается покупатель или поставщик. Выбирается из справочника "Контрагенты".
  • Налогообложение. Тип налогообложения продажи товаров и услуг налогом на добавленную стоимость.
  • Расшифровка платежа. Табличная часть или группа реквизитов, в зависимости от установленного представления ("Строкой" / "Списком"):
    • Договор. Указывается договор с контрагентом, по которому поступила оплата. Выбирается из справочника "Договоры".
    • Аванс. Признак аванса платежа.
    • Документ отгрузки (Документ расчетов), по которому осуществляется оплата (возврат оплаты). Для вида операции "От покупателя", указывается только для платежей, которые не являются авансом. Для вида операции "От поставщика" указывается всегда.
    • Сумма расчетов. Сумма оплаты. Указывается в валюте расчетов по договору с контрагентом.
    • Курс валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Кратность валюты расчетов по договору с контрагентом. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Сумма платежа. Сумма оплаты. Указывается в валюте кассы. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций.
    • Ставка НДС. Выбирается из справочника "Ставки НДС".
    • Сумма НДС. Указывается сумма НДС.
    • Оплачиваемый счет. Указывается в случае поступления (возврата) платежа по определенному счету. Выбирается из плана счетов "Управленческий".
    • Оплачиваемый заказ. Указывается в случае поступления (возврата) платежа по определенному заказу.
    • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Для вида операции "От подотчетника" становятся видимыми следующие реквизиты:

  • Подотчетник, от которого поступают денежные средства. Выбирается из справочника "Сотрудники".
  • Документ выдачи денежных средств подотчетнику.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Для вида операции "Розничная выручка" становятся видимыми следующие реквизиты:

  • Касса ККМ. Выбирается из справочника "Кассы ККМ".
  • Налогообложение. Тип налогообложения продажи товаров и услуг налогом на добавленную стоимость.
  • Ставка НДС. Выбирается из справочника "Ставки НДС".
  • Сумма НДС. Указывается сумма НДС.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".
 

Для вида операции "Розничная выручка (суммовой учет)" становятся видимыми следующие реквизиты:

  • Направление деятельности. При продажах на указанное направление деятельности будет относиться выручка, себестоимость и НДС с продаж.

  • Структурная единица - информация о розничной точке. Выбирается из справочника "Структурные единицы" с типом "Розничная точка (суммовой учет)".

  • Подразделение - информация о подразделении-исполнителе. Выбирается из справочника "Структурные единицы" с типом "Подразделение".

  • Налогообложение. Тип налогообложения продажи товаров и услуг налогом на добавленную стоимость.
  • Ставка НДС. Выбирается из справочника "Ставки НДС".
  • Сумма НДС. Указывается сумма НДС.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".
     

Для вида операции "Покупка валюты" становятся видимыми следующие реквизиты:

  • Операция - счет отнесения платежа.
  • Курс приобретения валюты. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций, и валюта учета соответствует национальной.
  • Кратность. Реквизит отображается на форме, если в настройках параметров учета включено ведение учета валютных операций, и валюта учета соответствует национальной.
  • Сумма в валюте учета.
  • Отражать разницу в курсе. Флаг устанавливается если курс по которому приобретается валюта, отличается от курса Центрального банка и необходимо отразить курсовую разницу в качестве доходов или расходов.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Для вида операции "Прочее" становятся видимыми следующие реквизиты:

  • Операция - счет отнесения платежа.
  • Документ планирования платежа. Выбирается из списка документов "Перемещение денег (план)" или Поступление денег (план)".

 

Документ "Поступление в кассу" может быть введен на основании документов:

  • "Акт выполненных работ",
  • "Расходная накладная",
  • "Продажа имущества",
  • "Счет на оплату",
  • "Перемещение денег (план)",
  • "Поступление денег (план)",
  • "Заказ покупателя",
  • "Отчет о розничных продажах".
