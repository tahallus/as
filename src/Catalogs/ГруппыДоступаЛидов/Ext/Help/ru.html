Группы доступа лидов

Справочник предназначен для описания доступа к элементам справочника Лиды по виду доступа Группы доступа лидов. Справочник используется в том случае, если необходимо ограничить доступ пользователям к конкретным лидам. Информация о том, к какой группе доступа принадлежит тот или иной лид, указывается в справочнике Лиды. Предусмотрена возможность отнесения лидов к определенной группе доступа с помощью сервисного механизма группового изменения реквизитов, который можно вызвать для выделенных лидов из формы списка справочника Лиды.

Важно. Установить ограничения по группам доступа лидов возможно только в том случае, если в разделе Настройки/Настройки пользователей и прав установлены опции Ограничивать доступ на уровне записей и Группы доступа лидов.
