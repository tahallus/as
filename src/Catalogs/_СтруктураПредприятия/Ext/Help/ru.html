Справочник "Структура предприятия" содержит управленческую структуру предприятия. Для каждого подразделения можно указать его руководителя.

Для открытия структура предприятия в разделе "Настройка и администрирование" в панели навигации выполните команду "Структура предприятия". Аналогичная команда доступна в других разделах.

Структура предприятия - это иерархический справочник с иерархией элементов (любое подразделение может иметь подчиненные подразделения). Для создания нового подразделения сначала выберите подразделение, в которое входит создаваемое, а затем выполните команду "Создать".

Для подразделения можно указать пользователей, входящих в подразделение. Для этого:

  • Командой "Все действия - Изменить" откройте форму подразделения;
  • Перейдите по ссылке "Пользователи" в панели навигации формы;
  • В открывшемся списке воспользуйтесь командой "Создать" для включения пользователя в текущее подразделение.

Для выполнения реорганизации структуры следует использовать команду "Все действия - Перенести в группу" или использовать механизм перетаскивания.
