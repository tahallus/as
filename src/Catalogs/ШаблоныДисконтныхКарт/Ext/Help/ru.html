Справочник «Шаблоны магнитных карт».

Список настроенных шаблонов магнитных дисконтных карт.

 

Для разработки шаблона рекомендуется использовать кнопку "Проверить код по шаблону":

  • Заполните необходимые поля шаблона.

  • Считайте код дисконтной карты с помощью считывателя магнитных карт.

  • Нажмите кнопку "Проверить код по шаблону".

  • Проверьте полученный код, который появится справа от кнопки. Если полученный код не верен, то повторите все пункты кроме п. 2 ещё раз.

    •  

    Для магнитных дисконтных карт предполагается, что данные записаны на 2-ой дорожке магнитной карты. Это наиболее распространенный вариант записи данных на магнитных дисконтных картах.
