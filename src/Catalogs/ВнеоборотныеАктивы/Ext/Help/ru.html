Справочник "Имущество"

Справочник предназначен для хранения перечня основных средств и нематериальных активов организации.

 

Редактирование информации о внеоборотном активе производится в отдельном диалоговом окне, открываемом из формы списка справочника имущества.

При этом указывается информация, характеризующая имущество:

  • Группа. Ссылка на группу справочника, в которую включен элемент. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую.
  • Тип имущества. Выбирается из списка: основное средство или нематериальный актив.
  • Способ амортизации имущества. Выбирается из списка: линейный или пропорционально объему продукции.
  • Единица измерения объема продукции. Заполняется при выбранном способе амортизации пропорционально объему продукции. Выбирается из справочника "Классификатор единиц измерения".
  • Начальная стоимость имущества в валюте учета.
  • Счет учета. Выбирается из Плана счетов "Управленческий". Требуется обязательное заполнение.
  • Счет амортизации. Выбирается из Плана счетов "Управленческий". Требуется обязательное заполнение.
