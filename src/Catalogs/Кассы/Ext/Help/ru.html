Справочник "Кассы"

Справочник предназначен для хранения перечня касс собственных организаций.

 

Редактирование информации о кассах производится в отдельном диалоговом окне, открываемом из формы списка справочника кассы.

При этом указывается информация, характеризующая кассу:

  • Валюта по умолчанию. Используется для автоматической подстановки в документы. Выбирается из справочника "Валюты".
  • Счет учета. Выбирается из Плана счетов "Управленческий". Требуется обязательное заполнение.
