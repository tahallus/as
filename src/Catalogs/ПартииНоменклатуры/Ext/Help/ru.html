Справочник "Партии номенклатуры"

Справочник предназначен для хранения информации о партиях запасов и организации партионного учета запасов.

Использование справочника зависит от параметра учета системы "Партии номенклатуры". Значение параметра устанавливается в разделе "Настройки" – "Еще больше возможностей" – "Общее".

Для номенклатурных позиций, принятых от сторонних контрагентов, учет по партиям обязателен. Для товаров, принятых на комиссию, должна быть заведена как минимум одна партия со статусом "Товары на комиссии"; для материалов, принятых в переработку, - "Давальческое сырье"; для запасов, принятых на ответственное хранение, - "Ответственное хранение".

Для элементов справочника доступен механизм установки значения партии "Основная" в рамках номенклатурной позиции. Для этого необходимо зайти в карточку справочника "Номенклатура", в навигационной панели перейти на вкладку "Партия", выбрать нужный элемент и нажать на кнопку "Использовать как основную". При добавлении первой партии в разрезе реквизитов "Статус" и "Владелец партии", если для номенклатуры установлены признаки "Использовать партии" и "Обязательное заполнение партии" и статус партии равен "Собственные запасы", данному элементу справочника автоматически будет установлен признак "Использовать как основную". Если установлен только признак "Использовать партии" и Статус партии равен "Давальческое сырье", "Товары на комиссии" или "Ответственное хранение", данному элементу справочника, также автоматически будет установлен признак "Использовать как основную". После выбора основной партии, при подборе номенклатуры в табличную часть документа, будет происходить автоматическое заполнение поля "Партия". Если для номенклатуры в документе могут использоваться партии с различными статусами(Пример: Расходная накладная. Статусы: Собственные запасы, Товары на комиссии), тогда партия автоматически установится только в том случае, если из всех доступных статусов партий для этого документа, признак "Основная" будет иметь только одна. Для документа "Расходная накладная", если у партии установлен реквизит "Владелец партии" который совпадает с значением поля документа "Поставщик" и она является основной в разрезе доступных статусов для операции документа, то данная партия будет автоматически установлена в соответствующую строку документа. В противном случае, будет установлена партия с признаком "Для всех владельцев"(Реквизит "Владелец партии" не заполнен) если у номенклатуры имеется "Основная" партия с данным значением.
