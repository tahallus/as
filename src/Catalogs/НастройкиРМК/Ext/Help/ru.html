Справочник «Настройки рабочего места кассира»

В справочнике хранятся настройки РМК. Один элемент соответствует одному элементу справочника «Рабочие места». Это позволяет хранить различные настройки для разных кассиров и компьютеров, за которыми они работают.

В справочнике хранится список быстрых товаров и список кнопок и команд меню.

По списку быстрых товаров создаются кнопки быстрых товаров в РМК. Для каждой кнопки можно задать заголовок, порядок вывода (поле сортировки) и комбинацию клавиш для нажатия кнопки с клавиатуры.

Список быстрых товаров можно заполнить элементами группы номенклатуры или скопировать их из другого элемента справочника «Настройки рабочего места кассира».

По списку кнопок и команд меню задаются заголовки и комбинации клавиш для ключевых кнопок и команд меню РМК. Список кнопок ограничен.
