Справочник "Источники привлечения"

Справочник предназначен для хранения информации об источниках привлечения клиентов.

Редактирование информации об источниках производится в отдельном диалоговом окне, открываемом из формы списка справочника источников привлечения.
При этом указывается информация, характеризующая источник:

  • Краткое наименование. Используется для выбора и быстрого поиска источника в списках. 
  • Группа. Ссылка на группу справочника, в которую включен элемент. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую

Для каждого источника в карточке можно указать e-mail и телефон, для того, чтобы отслеживать информацию с какого канала поступают обращения. При поступлении события на указанную контактную информацию, источник привлечения будет заполнен автоматически.
