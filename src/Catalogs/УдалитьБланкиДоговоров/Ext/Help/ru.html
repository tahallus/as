Справочник "Бланки договоров"

Справочник Бланки договоров предназначен для хранения шаблонов, используемых при формировании печатных форм договоров с контрагентами.

Поставляемые в комплекте с конфигурацией бланки можно редактировать или создавать на их основе собственные.

При помощи выбранного бланка, для каждого договора контрагента можно сформировать печатную форму.

При создании или редактировании бланка договора можно использовать предопределенные параметры, которые будут заполнены в печатной форме данными конфигурации автоматически. Также можно использовать заполняемые вручную поля, которые редактируются в тексте договора непосредственно перед печатью.
