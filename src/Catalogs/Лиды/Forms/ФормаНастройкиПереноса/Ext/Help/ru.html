Настройка соответствия полей лида и покупателя для переноса


Можно настроить список полей лида, данные из которых необходимо переносить в покупателя при конвертации. Для выбора доступны как основные реквизиты лида и контрагента, так и дополнительные.

Для настройки соответствия необходимо выбрать из какого поля лида в какое поле контрагента переносить данные. При выборе полей для настройки показываются только поля, совпадающие по типу значения.

Пример.

Для того, чтобы учитывать данные по ИНН для лида можно создать дополнительный реквизит ИНН(тип строка) и настроить его перенос в основной реквизит контрагента ИНН.
