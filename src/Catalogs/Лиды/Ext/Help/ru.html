Справочник "Лиды"

Справочник предназначен для хранения лидов, с которыми ведется работа в компании. Лиды - это потенциальные клиенты, проявляющие интерес к Вашей компании.

Редактирование информации о лидах производится в отдельном диалоговом окне, открываемом из формы списка справочника лидов.
При этом указывается информация, характеризующая лид:

  • Краткое наименование. Используется для выбора и быстрого поиска лида в списках.
  • Состояние. Информация о том, на каком этапе сейчас ведется работа с лидом.
  • Группа. Ссылка на группу справочника, в которую включен элемент. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую.
  • Ответственный. Информация о сотруднике, ответственном за работу с потенциальным клиентом.
  • Группа доступа. Группа доступа лидов, определяющая ограничение доступа пользователей к элементам справочника.

Когда предварительная работа с лидом завершена, его можно конвертировать в покупателя или, если намечается сделка, то одновременно в покупателя и заказ. Если лид оказался «некачественным» (например, если был пропущенный звонок, но человек просто ошибся номером или хотел совсем не то, что вы предлагаете), то необходимо отметить это отдельно. Необходимо указать причину завершения работы и если есть дополнительная информация поясняющая причину указать ее в комментариях.

Можно настроить поля, данные из которых нужно переносить при конвертации лида в покупателя. Для этого необходимо нажать на кнопку с изображением шестеренки в разделе "Результат". 

На закладке "Дополнительная информация" указывается любая дополнительная информация о лиде.
При нажатии кнопки "Дополнительные реквизиты" открывается диалоговое окно редактирования дополнительных реквизитов, определяемых пользователем.
