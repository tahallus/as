Форма позволяет указать виды минимальных цен для продажи.

Общие для всех - для всех документов продажи будут использоваться общие минимальные цены.
По структурным единицам - можно указать для каждой структурной единицы свои минимальные цены.

Документы, которые поддерживают работу с минимальными ценами:

Акт выполненных работ - только общий вид минимальных цен
Задание на работу - только общий вид минимальных цен
Корректировка реализации - склад указывается в табличной части, минимальные цены подбираются из склада
Отчет о переработке - склад указывается в шапке документа, минимальные цены подбираются из склада
Переоценка товаров на комиссии - только общий вид минимальных цен
План продаж - только общий вид минимальных цен
Чек ККМ возврат - только общий вид минимальных цен
Отчет комиссионера о списании - только общий вид минимальных цен
Передача товаров между организациями - склад указывается в табличной части, минимальные цены подбираются из склада
Расходная накладная - склад указывается в табличной части либо в шапке документа, минимальные цены подбираются из склада
Счет на оплату - склад указывается в шапке документа, минимальные цены подбираются из склада
Чек ККМ - склад указывается в шапке документа, минимальные цены подбираются из склада
Заказ наряд - склад указывается как в шапке так и в табличной части, минимальные цены подбираются из склада
Заказ покупателя - в зависимости от параметра "Резервирование запасов" склад может указываться в шапке, 
 табличной части, но может и отсутствовать, в таком случае минимальные цены применяются общие.

Если в документе указанная цена меньше минимальной, то текст поля цена будет выделен красным цветом.
