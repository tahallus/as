Параметры автоперемещение запасов

Дополнительный диалог "Параметры автоперемещение запасов" для заполнения и изменения данных об установке параметров автоперемещения открывается при нажатии гиперссылки "Просмотр и редактирование параметров автоперемещения запасов" в форме элемента справочника структурная единица организации.

При открытии диалога показываются установленные на текущий момент времени значения параметров. Количество и характер управляющих элементов в этом окне зависят от типа структурной единицы (Подразделение, Склад).

Это могут быть такие элементы, как:

  • Перемещение, комплектация, производство. Группа реквизитов:
    • Списать из - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, с которой будут списываться запасы. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Перемещение" в поля "Склад" и "Ячейка" при выборе значений в полях "Получатель" и/или "Ячейка (получатель)", в документ "Производство" на закладке "Запасы" в поля "Списывать из" и "Ячейка" при выборе значений в полях "Подразделение" и/или "Ячейка" (подразделения) и в документ "Заказ на комплектацию, производство" в поле "Склад резерва" при выборе значения в поле "Подразделение".
    • Получатель - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, в которую будут оприходованы запасы. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Перемещение" в поля "Получатель" и "Ячейка (получатель)" при выборе значений в полях "Склад" и/или "Ячейка" и в документ "Производство" в поля "Получатель" (получатель готовой продукции) и "Ячейка (получатель)" при выборе значений в полях "Подразделение" и/или "Ячейка" (подразделения).
    • Получатель отходов - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, в которую будут оприходованы возвратные отходы. Значение по умолчанию подставляется в документ "Производство" на закладке "Отходы" в поля "Получатель" и "Ячейка (получатель)" при выборе значений в полях "Подразделение" и/или "Ячейка" (подразделения).
  • Списание на расходы. Группа реквизитов:
    • Списать из - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, с которой будут списываться запасы на расходы. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Списание на расходы" в поля "Склад" и "Ячейка" при выборе значений в полях "Получатель" и/или "Ячейка (получатель)".
    • Получатель - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, на которую будут отнесены расходы. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Списание на расходы" в поля "Получатель" и "Ячейка (получатель)" при выборе значений в полях "Склад" и/или "Ячейка".

  • Передача в эксплуатацию. Группа реквизитов:

    • Списать из - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, с которой будут списываться запасы для передачи в эксплуатацию. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Передача в эксплуатацию" в поля "Склад" и "Ячейка" при выборе значений в полях "Получатель" и/или "Ячейка (получатель)".

    • Получатель - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, в которую будут оприходованы запасы в эксплуатацию. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Передача в эксплуатацию" в поля "Получатель" и "Ячейка (получатель)" при выборе значений в полях "Склад" и/или "Ячейка".

  • Возврат из эксплуатации. Группа реквизитов:

    • Списать из - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, с которой будут возвращены запасы из эксплуатации. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Возврат из эксплуатации" в поля "Склад" и "Ячейка" при выборе значений в полях "Получатель" и/или "Ячейка (получатель)".

    • Получатель - выбирается из справочника "Структурная единица". Указывается структурная единица и ячейка, в которую будут оприходованы возвращенные запасы из эксплуатации. Значение по умолчанию подставляется в документ "Перемещение запасов" для вида операции "Возврат из эксплуатации" в поля "Получатель" и "Ячейка (получатель)" при выборе значений в полях "Склад" и/или "Ячейка".
