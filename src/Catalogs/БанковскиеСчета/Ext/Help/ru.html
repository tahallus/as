Справочник "Банковские счета"

В справочнике "Банковские счета" хранятся банковские счета всех юридических и физических лиц: собственных и сторонних.

Справочник "Банковские счета" подчинен справочникам "Организации", "Контрагенты" и "Физические лица".

 

Список банковских счетов контрагента, собственной организации или физического лица определяется теми банковскими счетами, которые подчинены соответствующему элементу справочника "Организации",  "Контрагенты" или "Физические лица".

Для каждого контрагента или собственной организации можно выбрать т.н. банковский счет по умолчанию, который будет подставляться по умолчанию в платежные документы.

Аналитический учет денежных средств ведется по каждому банковскому счету.

 

Редактирование информации о банковском счете производится в отдельном диалоговом окне, открываемом из формы списка справочника банковских счетов.

При этом указывается информация, характеризующая банковский счет:

  • Банк. Заполнение справочника рекомендуется начинать с выбора банка. Банк влияет на отображение реквизитов на форме и состав проверок ввода. Выбор иностранного банка разрешает ввод банковского счета в форматах, предусмотренных для зарубежных банков.
  • Номер счета. Номер банковского счета.
  • Вид счета. Предлагается выбор из списка: расчетный, депозитный, ссудный, иной.
  • Валюта. Валюта хранения денежных средств на счете. Выбирается из справочника "Валюты".
  • Банк, в котором открыт счет. Группа реквизитов:
    • Корр. счет. Корреспондентский счет.
    • Город. Город банка.
  • Используется банк для расчетов. Установите флажок ""Используется банк для расчетов"", если банк, в котором открыт счет, проводит расчеты через корреспондентский счет, открытый в другом банке (банке для расчетов).
  • Банк для непрямых расчетов. Группа реквизитов:
    • Банк. Банк для расчетов.
    • Корр. счет. Корреспондентский счет.
    • Банк. Банк, в котором открыт счет.
    • Город. Город банка.
  • Настройки печати платежных поручений. Группа реквизитов:
    • Редактировать текст наименования организации. Устанавливает доступность для редактирования текста наименования организации. Аналогичный механизм используется для счетов контрагентов.
    • Текст наименования организации в поле "Плательщик". Текст, который будет подставляться в поле "Плательщик" в платежных документах. Аналогичный механизм используется для счетов контрагентов.
    • Вариант указания КПП. Определяет вариант указания КПП: либо во всех платежных поручениях, либо только при перечислении налогов. Аналогичный механизм используется для счетов контрагентов.
    • Вариант вывода месяца. Определяет вариант вывода месяца: либо строкой, либо числом.
    • Выводить сумму без копеек, если она в целых рублях. При установке данного флажка сумма в платежном документе будет указана без копеек, со знаком "=". В противном случае будет напечатан знак "-". Например, сумма 20 рублей в первом случае будет напечатана как 20=, а во втором случае как 20-00.
  • Дополнительная информация. Группа реквизитов:
    • Дата открытия. Дата открытия банковского счета.
    • Дата закрытия. Дата закрытия банковского счета.
    • Номер и дата разрешения. Номер и дата разрешения ЦБ
      на открытие валютного счета.
  • Панель навигации банковского счета.
    • Счет учета. Выбирается из Плана счетов "Управленческий". Требуется обязательное заполнение для банковских счетов организаций. Для контрагентов и физических лиц заполнять необязательно.

Для выбора банка достаточно заполнить поле БИК или Корр. счет. При вводе значения в одно их этих полей происходит поиск банка в справочнике и автоподстановка значений в остальные поля. В случае если банк с таким БИК или корр. счетом отсутствует в справочнике банков, можно выбрать банк самостоятельно из списка банков или ввести новый банк.
