Виды алкогольной продукции ЕГАИС

Справочник Виды алкогольной продукции ЕГАИС предназначен для хранения информации о видах реализуемой алкогольной продукции на предприятии.

Справочник доступен для использования, если включена функциональная опция Вести сведения для деклараций по алкогольной продукции.

Информация в справочнике заполняется из классификатора Виды алкогольной продукции ЕГАИС, который включен в поставку. Для заполнения справочника в соответствии с классификатором следует использовать команду Подобрать. Можно выделить несколько элементов в списке классификатора и добавить их в справочник.

При загрузке из ЕГАИС Товарно-транспортных накладных или Классификатора алкогольной продукции, элементы справочника создаются автоматически.

Если для вида продукции установлен флаг Маркируемый, то при розничной продаже появится запрос на сканирование акцизной марки.
