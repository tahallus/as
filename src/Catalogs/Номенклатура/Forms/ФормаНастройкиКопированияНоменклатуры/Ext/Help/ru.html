Копирование связанной информации

 

Форма позволяет выбрать связанную информацию для копирования при записи нового элемента справочника номенклатуры. К связанной информации относятся:

  • Единицы измерения
  • Характеристики (копируются только индивидуальные характеристики номенклатуры)
  • Спецификации
  • Состав наборов

В форме отображаются только разделы, в которых есть данные для копирования. Для каждого раздела можно выбрать режим копирования: не копировать, копировать все или копировать выборочно
