Справочник "Номенклатура"

Справочник предназначен для хранения информации о товарах, материалах, продукции, полуфабрикатах, объектах вложений в имущество, работах, услугах, технологических операциях и видах работ.

В списке номенклатуры можно отображать колонки с остатками и ценами, установить отборы по наличию или отсутствию на складе, диапазону цен, типу номенклатуры, категории или группе.

Чтобы быстро оформить документ из списка, выберите одну или несколько (удерживайте Ctrl) строк номенклатуры и используйте меню "Продать" или "Купить". Можно подобрать номенклатуру до необходимой суммы, перетаскивая нужные позиции в корзину, и затем оформить документ.

Для поиска номенклатуры в списке по штрихкоду можно использовать сканер, или ввести штрихкод вручную (F7). Курсор установится на строке списка найденной номенклатуры. 

Для организации практичной классификации справочника предлагается относить каждую номенклатурную позицию к соответствующей Категории, что позволит в дальнейшем пользоваться всеми удобствами фильтрации номенклатуры по свойствам.

Для вашего удобства мы предоставляем подробный классификатор Яндекс.Маркета с уже созданными свойстами номенклатуры и их популярными значениями.

Так же, справочник имеет многоуровневую иерархическую структуру. Элементы справочника можно перегруппировать, перенося их из группы в группу.

 

В данный справочник заносится следующая информация:

  • Тип. Тип номенклатуры определяет состав реквизитов элемента номенклатуры и порядок использования.
    • Запас. Материальные запасы, подлежащие складскому учету.
    • Работа. Подрядные работы производственного характера, например, строительство дома, пошив одежды.
    • Услуга. Услуги (сервис), например, туристическое обслуживание, услуги парикмахера, юридические услуги или услуги сторонних организаций, относящиеся на расходы компании, например, аренда помещения или оборудования.
    • Операция. Технологические операции, выполняемые в процессе выполнения работ, оказания услуг, производстве продукции.
    • Вид работ. Вид работ, на который может быть задана расценка.
  • Категория. Укрупненный классификатор номенклатуры, задается отдельным справочником в системе. Каждой категории могут быть назначены общие для всех входящих в нее позиций номенклатуры единицы измерения, свойства и характеристики.
  • Наименование. Используется для отображения в программе.
  • Наименование для печати. Используется для вывода в печатные формы документов. Например: Йогурт "Bio двойная польза" клубника шиповник 480 г - это полное наименование, Bio 2 польза, клубн.шип.480 йогурт - наименование для быстрого поиска.
  • Описание. Произвольное текстовое описание объекта номенклатуры.
  • Единица измерения. Каждая номенклатурная позиция характеризуется учетной единицей измерения. Она выбирается из справочника "Классификатор единиц измерения". Кроме базовой единицы измерения, позиция номенклатуры может иметь необходимое число других единиц измерения (упаковок). Например, сигареты могут иметь единицы измерения: «штука», «пачка», «блок», «коробка». Упаковки могут быть заданы как индивидуально для конкретных позиций номенклатуры в панели навигации элемента, так и для номенклатурной группы в целом. Каждая единица измерения (упаковка) имеет свой коэффициент пересчета относительно базовой единицы.
  • Артикул. Цифровой код товара. Текстовое значение, используется в печатных формах документов.
  • Группа. Ссылка на группу справочника, в которую включен элемент. При ручном изменении реквизита "Группа" происходит перемещение текущего элемента из старой группы в новую.
  • Код. Код элемента номенклатуры.
  • Весовой. Для ситуаций, когда используется обмен с подключаемым оборудованием Offline, предусмотре реквизит Весовой, чтобы отличать товары, которые должны выгружаться и нумероваться особым порядком на весах с печатью этикеток.
 
Для номенлатуры типа Запас могут быть назначены различные изображения. Добавить их можно следующими способами:

  • Изображение - Правая кнопка мыши - Добавить;
  • Через панель навигации перейти на вкладку "Файлы".

 

Учетная информация

  • Использовать характеристики. Признак того, что для данной номенклатуры ведется учет по характеристикам. Используется для номенклатуры типа: Запас, Работа и Услуга (только услуги выполняемые нашей организацией). Информация о характеристиках номенклатуры заносится в справочник "Характеристики номенклатуры". Для установки данного признака и просмотра списка характеристик необходимо перейти из панели навигации элемента на владку "Характеристики".
  • Обязательное заполнение характеристики. Признак отвечает за контроль обязательного заполнения характеристики в документах. Используется только при установленном признаке "Использовать характеристики". Для установки данного признака, необходимо перейти из панели навигации элемента, на владку "Характеристики".
  • Использовать партии. Признак того, что для данной номенклатуры ведется учет по партиям. Информация о партиях номенклатуры заносится в справочник "Партии номенклатуры". Для номенклатурных позиций, принятых от сторонних контрагентов, учет по партиям обязателен. Для товаров, принятых на комиссию, должна быть заведена как минимум одна партия со статусом "Товары на комиссии", для материалов, принятых в переработку, - "Давальческое сырье", для запасов, принятых на ответственное хранение, - "Ответственное хранение". Для установки данного признака и просмотра списка партий необходимо перейти из панели навигации элемента на владку "Партии".
  • Обязательное заполнение партии. Признак отвечает за контроль обязательного заполнения партий в документах. Используется только при установленном признаке "Использовать партии". Для установки данного признака, необходимо перейти из панели навигации элемента, на владку "Партии". 
  • Направление деятельности. При продажах товаров, работ и услуг на указанное направление деятельности будет относиться выручка, себестоимость и НДС с продаж. При отражении расходов, возникающих при поступлении услуг сторонних организаций, указанное направление деятельности будет использоваться по умолчанию для оформления документов.
  • Способ списания. Метод оценки запасов при перемещении, продаже, списании запасов. Доступные методы: "По средней" (взвешенная оценка), "FIFO" (взвешенная оценка). В оперативном учете запасы всегда списываются по средней цене (скользящая оценка, определяемая на момент документа). В регламентной процедуре закрытия месяца происходит корректировка стоимости списания согласно выбранному способу.
  • Срок исполнения (дн.). Стандартный срок выполнения заказа для обещания клиенту. Используется при оформлении заказов покупателей. Используется для номенклатуры типа: Запас, Работа и Услуга (только услуги, выполняемые нашей организацией).
  • Ставка НДС. Ставка НДС по умолчанию, для оформления документов.

Флажки ведения учета по партиям и характеристикам в форме номенклатуры появляются только в том случае, если на предприятии ведется учет по партиям и характеристикам, что указывается в Категории, к которой относится Номенклатурная позиция или в настройках учета (Раздел "Настройки" - "Настройка учета").

 

Набор / комплект

  • Это набор / комплект. Признак использования элемента номенклатуры в качестве набора. Доступен для номенклатуры с типами "Запас" и "Работа". Такой элемент можно использовать только при продаже, в других местах выбор его будет запрещен. Отметка разрешает редактирование следующих реквизитов.
  • Способ расчета цены набора. Возможен как обычный режим, когда цена будет складываться из цен комплектующих, так и установка собственной цены набора.
  • Способ вывода в печатных формах. Вариант отображения набора в печатных формах.
  • Состав набора. В таблице заполняется номенклатурный состав набора. Для набора с типом "Запас" возможно заполнение номенклатуры с типами "Запас" и "Услуга". Для набора с типом "Работа" возможно заполнение номенклатуры с типами "Работа" и "Услуга".  Для варианта расчета цены набора "Назначается за набор и распределяется по комплектующим пропорционально долям стоимости" отображается дополнительная колонка "Доля стоимости", в которой задается пропорция для расчета цен комплектующих в составе набора.

При использовании характеристик для номенклатуры-набора редактирование всей информации о наборе выполняется в отдельном окне. В таком случае можно заполнить свой состав для каждой характеристики.

Если номенклатура уже используется в составе других наборов, вместо вышеуказанных реквизитов выводится их список. Вложенное использование наборов запрещено.

 

Закупка и производство

  • Способ пополнения. Способ пополнения запасов номенклатурной позиции по умолчанию: закупка - приобретение у поставщика, производство - изготовление собственными силами, переработка - производство подрядчиком из материалов компании.
  • Поставщик. Поставщик позиции номенклатуры по умолчанию. Используется для номенклатуры типа: Запас и Услуга (только услуги сторонних организаций).
  • Страна происхождения. Страна, из которой ввезена позиция номенклатуры. Указывается для импортных товаров.
  • Спецификация. Спецификация изготовления номенклатурной позиции по умолчанию. Перейти к списку спецификаций можно из панели навигации элемента.
  • Срок пополнения (выполнения, дн.). Срок в днях, в течение которого возможно пополнение запаса позиции номенклатуры, выполнение работы или оказание услуги.
  • Норма времени (ч). Норма времени по умолчанию для операции.
  • Только поддержка остатка при расчете потребностей. При установке опции, в расчете потребностей для этого запаса не будут анализироваться данные заказов. Учитываются только данные об остатках, указанные в управлении запасами.
