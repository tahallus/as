Справочник "Параметры расчетов доставки"

Справочник предназначен для хранения настроенных правил выборки данных из различных разделов учета, которые впоследствии будут использоваться при стоимости или себестоимости доставки.

 

Каждый элемент справочника фактически содержит шаблон запроса для получения определенного значения из базы данных. Запрос может быть составлен двумя способами. Если установлен флажок "Произвольный запрос", то для редактирования доступны текст и параметры запроса, в толстом клиенте доступен конструктор запроса. Если флажок не установлен, то необходимо указать источник данных, периоды отбора, показатель (один), установить отборы данных. В этом случае запрос составится автоматически. Если установлен флажок "Задавать значение при расчете", то элемент не требует настройки; значение параметра необходимо будет задать перед расчетом в документе "Заказ покупателя".

Значения параметров запроса могут заполняться несколькими способами:

  1. Могут быть заданы фиксированные значения параметров запроса на закладке "Отборы" или, если установлен флажок "Произвольный запрос", на закладке "Параметры".
  2. Могут быть использованы следующие фиксированные параметры запроса: Заказ, ДатаЗаказа, Контрагент и Запасы. Значения этих параметров будут автоматически подставлены из данных документа "Заказ покупателя".

Необходимо заполнить следующие реквизиты:

  • Идентификатор - представление параметра расчетов в формуле.
  • Источник выборки данных.
  • Задавать значение при расчете - если значение параметра будет вводиться непосредственно перед расчетом в заказе покупателя.
  • Произвольный запрос. Флажок устанавливается, если запрос и параметры запроса будут редактироваться вручную.

На закладке "Данные" указывается период выборки данных и отмечается показатель.

На закладке "Отбор" накладываются отборы на выборку данных.

На закладке "Запрос" можно просмотреть результат настройки параметра расчетов и при необходимости отредактировать его. Редактирование запроса доступно при нажатой кнопке "Редактировать запрос". В толстом клиенте можно воспользоваться конструктором запроса по кнопке "Конструктор запроса".

  • Запрос - текст запроса.
  • Параметры - параметры запроса:
    • Имя параметра. Подставляется из текста запроса, не содержит пробелов.
    • Значение параметра.
