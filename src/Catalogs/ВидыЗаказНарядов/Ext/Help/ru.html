Справочник "Виды заказ-нарядов"

Справочник предназначен для группировки различных состояний заказ-нарядов по бизнес-процессам.

Каждый вид заказа будет представлять собой отдельную воронку продаж.

Для ввода дополнительных видов требуется включение функциональной опции "Виды заказ-нарядов".

В реквизите "Состояние выполнения" выбирается состояние, начиная с которого заказ-наряды будут выполнять расходные движения: отгружать товары со склада, начислять зарплату и т.д.
