Настройки рассылки Состояние компании

Справочник используется для настройки механизма регулярной рассылки сводного отчета о состоянии компании.

Имеется возможность настройки расписания. Можно изменить порядок и состав секций отчета, изменить заголовки секций, наложить отборы.

По кнопке Отправить прямо сейчас будет выполнена тестовая отправка рассылки по электронной почте.

По кнопке Предварительный просмотр отчет будет показан на экране.

При выводе страниц отчета выполняется контроль, чтобы, по-возможности, секции умещались на страницу. В противном случае, в отчет вставляется разрыв страницы.

По каждой секции отчета существует возможность настроить индивидуальные фильтры. Фильтры могут быть настроены как по полям запроса, так и по вложенным реквизитам полей. Можно накладывать фильтры «В группе». При применении нескольких однотипных фильтров с одинаковым полем, например «СтруктурнаяЕдиница», условия объединяются по «ИЛИ». Условия для фильтров с различными полями, например «Номенклатура» и «Контрагент», объединяются по «И».

По кнопке "Ещё" доступны следующие команды:

Скопировать - для добавления варианта рассылки,
Пометить на удаление - для отключения варианта рассылки,
Показать в списке - для перехода к списку всех вариантов рассылки,
Настройки - для настройки даты предварительного просмотра и учетной записи, от имени которой будет совершаться рассылка.

Для отключения задания рассылки можно либо пометить элемент справочника на удаление, либо очистить поле Учетная запись, либо очистить все поля Получатель.
