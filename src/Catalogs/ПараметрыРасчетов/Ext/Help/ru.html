Справочник "Параметры расчетов"

Справочник предназначен для хранения настроенных правил выборки данных из различных разделов учета, которые впоследствии будут использоваться при расчете заработной платы.

 

Каждый элемент справочника фактически содержит шаблон запроса для получения определенного значения из базы данных. Запрос может быть составлен двумя способами. Если установлен флажок "Произвольный запрос", то для редактирования доступны текст и параметры запроса, в толстом клиенте доступен конструктор запроса. Если флажок не установлен, то необходимо указать источник данных, периоды отбора, показатель (один), установить отборы данных. В этом случае запрос составится автоматически. Если установлен флажок "Задавать значение при расчете зарплаты", то элемент не требует настройки; значение параметра необходимо будет задать перед расчетом в документе "Начисление зарплаты".

Значения параметров запроса могут заполняться несколькими способами:

  1. Могут быть заданы фиксированные значения параметров запроса на закладке "Отборы" или, если установлен флажок "Произвольный запрос", на закладке "Параметры".
  2. Значения параметров запроса подставляются автоматически из документа "Начисление зарплаты". К таким параметрам относятся:  Организация, Валюта, Подразделение, СтруктурнаяЕдиница, ПериодРегистрации, МоментВремени, НачалоПериода, КонецПериода, Сотрудник, ТипЗанятости, ТабельныйНомер, ТабНомер, Исполнитель, ФизЛицо, ФизическоеЛицо, Должность, ВидНачисленияУдержания, ЗаказПокупателя, Заказ, Проект, СчетЗатрат, НаправлениеДеятельности, Размер, ОтработаноДней, ОтработаноЧасов. Задавать значения перечисленных параметров не нужно. Заданные значения на расчет не повлияют.

Необходимо заполнить следующие реквизиты:

  • Идентификатор - представление параметра расчетов в формуле. Должен начинаться с буквы и не содержать служебные символы. 
  • Источник выборки данных.
  • Задавать значение при расчете зарплаты - если значение параметра будет вводиться непосредственно при расчете зарплаты.
  • Произвольный запрос. Флажок устанавливается, если запрос и параметры запроса будут редактироваться вручную.

На закладке "Данные" указывается период выборки данных и отмечается показатель.

На закладке "Отбор" накладываются отборы на выборку данных.

На закладке "Запрос" можно просмотреть результат настройки параметра расчетов и при необходимости отредактировать его. Редактирование запроса доступно при нажатой кнопке "Редактировать запрос". В толстом клиенте можно воспользоваться конструктором запроса по кнопке "Конструктор запроса".

  • Запрос - текст запроса.
  • Параметры - параметры запроса:
    • Имя параметра. Подставляется из текста запроса, не содержит пробелов.
    • Представление параметра. Представление отображается в документе "Начисление зарплаты".
    • Значение параметра.

 

Элемент справочника можно заполнить по шаблону, для этого необходимо нажать на кнопку "Заполнить по шаблону" и выбрать один из вариантов:

  • Норма дней;
  • Норма часов;
  • Отработано дней;
  • Отработано часов;
  • Отработано часов по заданиям;
  • Тарифная ставка;
  • Фиксированная сумма.

После выбора варианта элемент справочника будет настроен в соответствии с выбранным шаблоном. Далее элемент можно дополнить своими настройками.
