Справочник "Спецификации"

Справочник предназначен для хранения информации о спецификациях изготовления продукции, выполнения работ, оказания услуг.

Редактирование информации о составе спецификации производится в отдельном диалоговом окне, открываемом из формы списка справочника.

При этом указывается информация, характеризующая спецификацию:

  • Характеристика продукции. Указывается в случае, если спецификация характеризует вариант изготовления изделия по определенной характеристике.
  • Вид производства. Содержит порядок этапов производства спецификации.
  • Базовая спецификация. Отображается при включении настройки программы "Использовать параметрические спецификации" для спецификаций, созданных на основании параметрических.
  • Заказ. Заказ покупателя или заказ на производство, для которого была создана спецификация.
  • Комментарий. Произвольный комментарий к спецификации. 
  • Недействительна. Отметка устаревших и не использующихся спецификаций.
  • Состав. Табличная часть:
    • Тип строки. Тип строки состава изделия. Может принимать следующие значения:
      • Материал. Сырье или материал, закупаемые у сторонних контрагентов (поставщиков).
      • Сборка. Сборочная единица, полуфабрикат, производимый на предприятии.
      • Узел. Несуществующая единица, набор, состоящий из нескольких номенклатурных позиций, фантом. 
      • Расход. Услуга сторонних контрагентов производственного характера.
    • Этап. При длительном производстве указывает на каком этапе используется материал. Доступен только если заполнен вид производства в шапке спецификации.
    • Номенклатура. Номенклатурная позиция, входящая в состав изделия, в соответствии с указанным типом строки состава.
    • Характеристика. Характеристика номенклатурной позиции.
    • Единица измерения. Единица измерения номенклатурной позиции.
    • Количество. Норма расхода номенклатурной позиции на изготовление изделия.
    • Количество продукции. Количество продукции, на которое указаны нормы расхода номенклатурной позиции.
    • Доля стоимости. Доля стоимости запаса, получаемого в результате разборки (разделки) от стоимости исходной номенклатуры.
  • Операции. Табличная часть:
    • Этап. При длительном производстве указывает на каком этапе выполняется операция. Доступен только если заполнен вид производства в шапке спецификации.
    • Операция. Технологическая операция процесса изготовления изделия.
    • Норма времени. Норма времени на выполнение операции (в часах).
    • Количество операций. Количество операций, требуемых для производства продукции / выполнения работ. Доступно только для операций с расчетом по фиксированной стоимости. 
    • Количество продукции. Количество продукции, на которое указаны нормы времени выполнения операции.

При включении настройки программы "Использовать параметрические спецификации" появляется дополнительная возможность использования формул в составе спецификации для колонок "Номенклатура" и "Количество" и в таблице операций для колонок "Операция", "Норма времени", "Количество операций". Формулы в такой спецификации будут рассчитаны в момент проведения или расчета калькуляции заказа покупателя и проведения, заполнения таблицы материалов или операций заказа на производство. При расчете создается новая спецификация, привязанная к заказу, в котором она используется.
Для спецификаций с формулами игнорируется обязательность заполнения дополнительных реквизитов. Проверка их заполнения выполняется в момент расчета.
