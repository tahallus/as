Справочник "Сотрудники"

Справочник предназначен для хранения информации о сотрудниках организации.

Особенностью учета персонала является тот факт, что один и тот же человек (иначе физическое лицо) может рассматриваться как несколько объектов учета. Например, некое физическое лицо может поработать в организации и уволиться, а затем по прошествии некоторого времени снова быть принятым на работу. В этом случае два сотрудника, принятые в разное время и в общем случае на разные рабочие места, соответствуют одному физическому лицу. Таким образом, в рамках учета персонала мы сталкиваемся с двумя классами тесно связанных объектов учета: сотрудниками и физическими лицами.

 

Редактирование информации о сотруднике производится в отдельном диалоговом окне, открываемом из формы списка справочника сотрудников.

При этом указывается информация, характеризующая сотрудника:

  • Тип занятости сотрудника: основное место работы или совместительство.
  • Физ. лицо. Указывается физическое лицо, соответствующее данному сотруднику. Выбирается из справочника "Физические лица".
  • Счета учета расчетов. Группа реквизитов:
    • С персоналом. Указывается счет расчетов с персоналом. Выбирается из Плана счетов "Управленческий".
    • С подотчетником. Указывается счет расчетов с подотчетником. Выбирается из Плана счетов "Управленческий".
    • По перерасходу. Указывается счет расчетов по перерасходу. Выбирается из Плана счетов "Управленческий".

 

При создании нового элемента справочника предоставляется выбор:

  • Создать нового сотрудника и внести его личные данные в справочник физических лиц: в этом случае кроме сотрудника будет создан еще и элемент справочника "Физические лица" и установлена связь между ними.
  • Создать нового сотрудника, выбрав его из справочника физических лиц: в этом случае для выбора доступно физическое лицо, при выборе заполняется наименование сотрудника и устанавливается связь с личными данными.
  • Создать нового сотрудника: создается только элемент справочника "Сотрудники".

Если при создании нового сотрудника заполнить все поля, подчеркнутые красным, на закладке Данные о приеме на работу, то будет создан документ "Прием на работу".

После записи сотрудника личные данные и данные о приеме на работу больше недоступны для изменения. Их можно отредактировать, открыв созданные элемент справочника "Физические лица" и документ "Прием на работу".
