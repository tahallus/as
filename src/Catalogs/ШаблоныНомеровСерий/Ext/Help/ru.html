Справочник "Шаблоны серий"

Справочник "Шаблоны серий" предназначен для хранения списка всех тех шаблонов, которые применяются для генерации номеров серий в программе. Шаблоны этикеток и ценников могут быть добавлены непосредственно в форме подбора серий.

Пример:

Наберите в поле шаблона символы в том порядке, в каком они будут вводиться для серии
•  Укажите # вместо цифр номера серии, которые будут изменяться
•  Укажите префикс, суффикс или другие символы номера серии в порядке их следования в серии
Например: Для номера АА-000001 шаблон будет выглядеть так:АА-######
