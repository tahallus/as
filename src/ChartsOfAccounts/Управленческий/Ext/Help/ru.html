План счетов "Управленческий"

План счетов управленческого учета. Названия счетов и суть хранимой на них информации полностью определяются управленческой политикой организации.

При необходимости пользователь может в режиме "1С:Предприятие" добавлять новые счета и субсчета, устанавливая следующие их свойства:

  • настройка валютного учета,
  • признаки активных, пассивных и активно-пассивных счетов,
  • признаки забалансовых счетов,
  • типы счетов.

 

Анализ данных по счетам учета осуществляется с помощью отчетов:

  • "Оборотно-сальдовая ведомость".

 

Как добавить субсчета для существующего счета:

  1. Сделать счет субсчетом, т. е. изменить у него код, порядок и наименование как для субсчета.
  2. Ввести новый счет-группу, т. е. указать у него код и порядок, как у счета-родителя, а также указать тип счета - "Группа счетов".
  3. У субсчета (п. 1) в поле "Подчинен счету" указать счет-группу, созданный в п. 2.
  4. При необходимости завести дополнительные субсчета у счета-группы.

 

Пример: требуется детализировать информацию по незавершенному производству по материальным затратам и затратам на оплату труда рабочих. Для этого у счета "Незавершенное производство" необходимо завести два субсчета: "Материальные затраты" и "Зарплата производственных рабочих". Порядок внесения изменений:

  1. Открыть счет "Незавершенное производство", изменить код "20" на "20.01", порядок " 20" на " 20.01", наименование "Незавершенное производство" на "Материальные затраты".
  2. Добавить новый счет, указать у него тип - "Группа счетов", код - "20", порядок - " 20" и наименование - "Незавершенное производство".
  3. У субсчета "Материальные затраты", в поле "Подчинен счету", указать счет "Незавершенное производство".
  4. Добавить новый субсчет, указать у него тип - "Незавершенное производство", код - "20.02", порядок - " 20.02" и наименование - "Зарплата производственных рабочих", в поле "Подчинен счету" указать счет "Незавершенное производство".
