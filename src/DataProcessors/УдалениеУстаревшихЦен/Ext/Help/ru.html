Обработка "Удаление устаревших цен"

Обработка предназначена для удаления устаревших цен по условию.

 

Варианты удаления цен:

  • С выключенной опцией "Используется в прайс-листах";
  • Оставить последних несколько цен;
  • Почистить цены раннее указанной даты;

 

С выключенной опцией "Используется в прайс-листах".

В данном варианте будут удалены все записи о ценах номенклатуры, в которых выключена опция "Использовать запись в прайс-листах". Опция располагается в карточке цены.

 

Кроме последних несколько цен.

Для каждой номенклатуры (и характеристики) может быть введено неограниченное количество цен, при этом в работе необходимы только последние 1 или 2 записи. Эта настройка позволяет избавиться от устаревших записей и для каждой номенклатуры сохранить только N актуальных цен, где N задается пользователем.

 

Раннее указанной даты.

Для каждой номенклатуры (и характеристики) будут удалены цены, дата действия которых старше указанной.

 

ВАЖНО! Рекомендация по очистке цен не является обязательной, т.к. для каждой программы есть много факторов, которые влияют на скорость работы с ценами, например, наличие выделенного сервера, характеристики процессора, объем оперативной памяти, количество номенклатуры и т.д. Учесть все факторы сложно, поэтому этот совет носит только рекомендательный характер.

 

Удаление цен записывается журнал и по каждой результативной операции формируется файл, который, в последствии, прикладывается в папку ""Удаленные цены"" из хранилища файлов.

 

Перед удалением цен рекомендуем убедится в отсутствии конкурирующих процессов, которые обращаются к ценам. Примером таких процессов может быть загрузка или формирование цен, наличие не рассчитанных динамических цен, проведение документов вида Приходная накладная и т.д.
