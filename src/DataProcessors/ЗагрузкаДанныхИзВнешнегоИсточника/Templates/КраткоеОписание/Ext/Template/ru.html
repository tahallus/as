ЗАГРУЗКА ДАННЫХ ИЗ ВНЕШНЕГО ИСТОЧНИКА

 

Какие данные можно загружать и куда?

 

Поддерживается импорт из форматов xlsx, mxl, csv. Ищите на форме кнопку загрузки:

  • Табличные части документов Заказ покупателя, Заказ поставщику, Приходная и Расходная накладная, документы комиссионной торговли, справочника Спецификации и другие.
  • Справочники Номенклатура, Контрагенты, Контакты, Лиды;
  • Регистр сведений Цены номенклатуры;
  • Остатки товаров в документ Ввод начальных остатков - при загрузке из файла с фиксированной структурой

ВНИМАНИЕ! Операции загрузки данных в справочники, документы и регистры являются необратимыми! Т.е. в случае загрузки ошибочных данных (неактуальной информации) справочник, или регистр сведений, придется вычищать самостоятельно.

 

 

Как воспользоваться загрузкой?

 

Команды начала работы с механизмами доступны пользователям непосредственно из объектов. Так, например, что бы загрузить данные в табличную часть достаточно открыть соответствующий документ, а для загрузки элементов справочников вам необходимо перейти в форму списка. Загрузка цен осуществляется непосредственно из обработки Прайс-лист.

 

Ссылки для быстрого перехода:

  •  Справочник "Контрагенты"
  •  Справочник "Номенклатура"
  •  Регистр сведений "Цены" 

ВНИМАНИЕ! Загрузка данных доступна только пользователям с правами администратора приложения.

 

 

Пример загрузки данных в табличную часть документа Заказ поставщику.

 

1. Для загрузки данных из внешнего источника необходимо открыть документ и выбрать соответствующую команду

 

 

2. Указать файл

 

3. Для данных из файла необходимо указать каким колонкам они соответствуют

 

 

4. Проверить результат сопоставления, возможно, поправить данные

 

5. Загрузить данные в табличную часть документа

 

 

Загрузка данных копированием.

 

В основных объектах конфигурации доступна загрузка данных копированием. При использовании данного метода пользователю предоставляется возможность перенести данные в приложение 1с копированием/вставкой содержимого колонок документов.
