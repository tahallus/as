Обработка "Загрузка данных из внешних источников"

С помощью этой обработки можно загрузить данные из внешних источников (файлов), например, из электронных таблиц Excel или табличных документов 1с.

Получить наглядное представление о возможностях загрузки данных поможет видеоролик http://www.youtube.com/watch?v=PwcoBPi9Slc

Добавлена возможность при начале работы в программе загружать номенклатуру, категории, свойства, картинки, цены и остатки в одном файле с фиксированной структурой. Поддерживается возможность указания до 10 свойств товаров, имена которых нужно заполнить в колонках файла.

Добавлена возможность загружать картинки номенклатуры из локального каталога, или по ссылке в интернете. Для пользователей 1С-Fresh доступна загрузка только по ссылке. Файл картинки должен быть доступен по политике CORS, адрес к файлу должен быть указан в том же протоколе, который используется по-умолчанию на сайте (http или https).

Добавлена возможность загружать группы номенклатуры с иерархией, одним из двух способов:

  • Указывать группы номенклатуры в строках файла, с заполнением в колонке "Это группа" = "Да". Группы должны идти в файле раньше, чем входящие в них элементы. У группы так же может быть заполнено поле Родитель, тогда группа будет создана внутри указанного родителя (другой группы).
  • Указывать полный путь к элементу номенклатуры в виде иерархии с разделителями: ";" ">" "/" "\" в колонке Родитель для строки элемента номенклатуры. При загрузке будут сопоставлены или созданы все группы в иерархии, и в последнюю группу помещен элемент номенклатуры.
Если в колонке Родитель указано наименование группы-родителя, будет выполнен поиск такой группы, но сама группа создана не будет. Чтобы создать группу, нужно добавить её в начало файла отдельной строкой, указав в колонке ЭтоГруппа = "Да" и Наименование в соотв. колонке. Второй способ - указать группу как часть иерархии: "\ИмяГруппы". В начале строки иерархии должен быть символ-разделитель, один из ";" ">" "/" "\"

При загрузке номенклатуры можно загрузить информацию о штрихкоде.
В одной ячейке можно указать несколько штрихкодов. Для этого каждый штрихкод должен начинаться с новой строки.
