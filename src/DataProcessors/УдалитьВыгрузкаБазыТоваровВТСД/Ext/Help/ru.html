Обработка "Выгрузка базы товаров в ТСД"

Обработка предназначена для выгрузки данных о товарах в базу терминала сбора данных.

  • Для заполнения списка товаров используется кнопка Заполнить. В список заполняются только те товары, на которые назначены штрихкоды.
  • Для выгрузки товаров в терминал сбора данных используется кнопка Выгрузить. Выгрузка информации о товарах производится в том случае, если терминал сбора данных подключен и настроены параметры выгрузки (формат файла выгрузки, путь к файлу выгрузки).
