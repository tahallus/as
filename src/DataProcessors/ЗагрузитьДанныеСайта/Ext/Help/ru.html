Обработка "Загрузка данных сайта"

Позволяет загрузить данные с сайта, использующего CMS 1С-UMI или 1С-Битрикс и настроить интеграцию с сайтом для последующего обмена по стандарту CommerceML.

Укажите адрес сайта, имя пользователя и пароль. В дополнительных настройках можно указать организацию, загружать ли в 1С картинки с сайта, каталог для полученных с сайта товаров, вид цен, который будет использоваться для загруженных с сайта цен.

Для 1С-UMI:

Дополнительных настроек не требуется.

Посмотрите видео: https://youtu.be/uicE-ZEtuBs?t=2m50s

Для 1С-Битрикс:

Интеграция с 1С доступна для модулей сайта с указанными версиями (или новее):
Информационные блоки 17.0.5
Торговый каталог 17.0.4

Перед выгрузкой данных сайта в 1С выполните (проверьте) настройки CMS:

В CMS "1С-Битрикс: Управление сайтом" перейдите в раздел Администирование - Магазин - Настройки - Интеграция с 1С
или по ссылке: имя_сайта/bitrix/admin/1c_admin.php

На закладке Каталог установите:

  • Тип инфо-блока - Создать по необходимости
  • Загружать торговые предложения (характеристики) в отдельный инфоблок - установите галочку, если используете предложения с разными свойствами для товара (характеристики товара)

На закладке Экспорт каталога установите:

  • Инфоблок каталога - выберите каталог, который нужно выгрузить в 1С
  • Количество элементов выгружаемых за один шаг (0 - выполнять выгрузку за один шаг): установите 20

Ограничения для 1С-Битрикс:

В 1С не будут загружены: свойства товаров, типы которых не совместимы с 1С: с множественными значениями, с типом Файл, кроме картинок для товаров и предложений. При обмене с 1С несовместимые свойства на сайте останутся как есть.

Настройка узла обмена с сайтом

В 1С будет автоматически настроен обмен данными с сайтом в разделе CRM - Сайт компании - Обмен с сайтом - выберите узел обмена с сайтом.
Можно использовать интеграцию с сайтом в разных режимах:

  1. Интеграция каталога товаров и предложений с сайтом
    Этот вариант будет настроен после загрузки по умолчанию. Каталог товаров, цены и остатки товаров вводятся в 1С. При обмене с сайтом товары и предложения передаются на сайт, где обновляют имеющиеся данные или создают новые.
    Заказы с сайта загружаются в 1С.
  2. Обновление цен и остатков из 1С на сайте
    Чтобы использовать этот вариант, в 1С в узле обмена с сайтом, на закладке Выгрузка товаров выберите Использование обмена - Обновление на сайте цен и остатков товаров.
    При такой настройке, у существующих на сайте товаров будут обновляться предложения (цены и количество), но сами товары на сайте изменяться не будут.
    Заказы с сайта загружаются в 1С.
  3. Только загрузка заказов с сайта
    Товары и предложения вводятся на сайте, в 1С загружаются только заказы покупателей. При загрузке заказов с сайта, создаются покупатели с контактными данными и товары по данным заказа.
    Если товары были ранее загружены с сайта, они будут использованы в заказах.
    Чтобы использовать этот вариант, в 1С в узле обмена с сайтом, на закладке Основные настройки снимите галочку Выгрузка товаров, оставьте галочку Обмен заказами.

Настройки CMS после выполнения обмена с 1С-Битрикс:

При выгрузке предложений из 1С на сайт, будет создан новый пакет предложений в 1С Каталоге, предыдущие предложения останутся без изменений.
Если вы используете торговые предложения на сайте, для отображения переданных из 1С предложений понадобится настройка, выполнить её достаточно один раз:
В CMS перейти в Настройки - Настройки продукта - Настройки модулей - Торговый каталог
имя_сайта/bitrix/admin/settings.php?lang=ru&mid=catalog&mid_menu=1
Необходимо подключить к основному каталогу товаров новые предложения из 1С Каталога вместо старых предложений: снять галочку у информационного блока с типом [offers] со старыми предложениями и установить галочку для информационного блока с типом [1c_catalog] с новыми предложениями.
Для этого нужно перейти на закладку Каталоги, снять галочку у информационного блока с типом [catalog], сохранить, снять галочку у информационного блока с типом [offers], который нужно отключить, вернуть галочку у информационного блока с типом [catalog], сохранить.
Далее нужно настроить отображение новых предложений на сайте: для этого перейдите в раздел Сайт, включите режим правки, выделите область Каталог и настройте Параметры компонента согласно вашим требованиям.
