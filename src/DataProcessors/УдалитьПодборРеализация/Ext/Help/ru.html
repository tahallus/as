Обработка "Подбор товаров в документы реализаций"

Обработка используется для подбора номенклатуры (запасов, услуг) в незаполненную табличную часть документа. Обработка вызывается в документах, имеющих табличную часть, по кнопке Подобрать. Подбор осуществляется из справочника Номенклатура, открывающегося в режиме подбора.

ВАЖНО! Основные функции работы со списками вызываются из контекстного меню.

Возможности списков и контекстных меню

Ключевым справочником для механизма подбора является номенклатура. Данный справочник хранит записи о запасах, услугах и прочих номенклатурных позициях, использующихся в хозяйственной деятельности. Именно по отношению к данному справочнику в обработке подбора будет строиться и показываться дополнительная информация. Обработка подбора предоставляет следующую дополнительную информацию по номенклатуре:

  • список характеристик (если включена соответствующая опция);

  • цены номенклатуры по виду цен, указанному в документе, из которого произведен вызов подбора. Посмотреть информацию о текущем виде цен можно, вызвав команду Сведения о документе;

  • остатки запасов, свободные остатки запасов и указание наличия резерва (если включено использование резервирования запасов). Для запасов, по которым присутствует резерв, есть возможность посмотреть расшифровку резерва: какими документами оформлялось резервирование (команда вызывается из контекстного меню);

     

    Описание списков и контекстных меню

    Основная информация представлена в виде списков. Все вспомогательные команды размещаются в контекстном меню (вызов правой клавишей мышки) и для удобства разбиты на группы. Ниже приведен перечень команд, доступных из контекстных меню.

     

    Список номенклатуры

  • Добавить товар – добавление товара в корзину

  • Создать – команда создания нового элемента справочника Номенклатура

  • Изменить – редактировать текущий элемент справочника Номенклатура

  • Скопировать – создать новый элемент справочника Номенклатура, скопировав в него значения из текущего элемента

  • Перейти к родителю – в списке иерархии номенклатуры позиционироваться на папке, которая является родителем для указанного элемента справочника Номенклатура. После позиционирования произойдет отбор в списке номенклатуры: пользователю будут доступны только позиции, которые входят в текущую папку

  • Расшифровать резерв – открыть список документов расшифровки резерва

  • Аналоги номенклатуры – в списке номенклатуры будут показаны только текущая номенклатура и ее аналоги (аналоги номенклатуры указываются непосредственно из карточки элемента справочника Номенклатура)

  • Создать/изменить цену – позволяет быстро создать/изменить текущую цену по указанной позиции

  • Обновить – обновить данные списка номенклатуры

  • Настройка списка – команда вызывает настройки динамического списка

  • Вывести список – позволяет вывести содержимое списка номенклатуры в табличный документ (используется, например, для печати или сохранения списка)

  • Изменить форму – позволяет изменять видимость, размер, порядок и прочие свойства элементов формы подбора

     

    Список характеристик номенклатуры

  • Добавить товар – добавление товара в корзину с текущей характеристикой

  • Создать – команда создания нового элемента справочника Характеристики номенклатуры

  • Изменить – редактировать текущий элемент справочника Характеристики номенклатуры

  • Показать номенклатуру – перейти (вернуться) к списку номенклатуры

  • Расшифровать резерв – аналогична команде из списка номенклатуры

  • Создать/изменить цену – аналогична команде из списка номенклатуры

  • Обновить – обновить данные списка характеристик номенклатуры

  • Настройка списка – команда вызывает настройки динамического списка характеристик номенклатуры

  • Вывести список – позволяет вывести содержимое списка характеристик номенклатуры в табличный документ (используется, например, для печати или сохранения списка)

  • Изменить форму – аналогична команде из списка номенклатуры

     

    Список складских остатков (если включено использование нескольких складов)

  • Обновить – обновить данные списка складских остатков

  • Вывести список – позволяет вывести содержимое списка складских остатков в табличный документ (используется, например, для печати или сохранения списка)

  • Изменить форму – аналогична команде из списка номенклатуры

     

    Список отобранной номенклатуры (Корзина)

    Отобранная номенклатура является табличной частью обработки подбора, поэтому команды контекстного меню по своему составу и назначению аналогичны командам табличных частей, используемых в документах.

     

    Навигация подбора и горячие клавиши

    Форма подбора обладает командной панелью, в которой собраны дополнительные возможности управления формой. К таким возможностям относятся:

  • управление видимостью списков: складские остатки, корзина

  • быстрый переход по элементам формы с указанием горячих клавиш. При подборе товаров можно использовать следующие горячие клавиши:

    Alt+F2 перейти к списку номенклатуры или списку характеристик номенклатуры

    Alt+F3 переход к строке полнотекстового поиска

    Alt+F5 перейти к списку иерархии номенклатуры

    Alt+F6 перейти к списку подобранных товаров

    Переходы также можно осуществлять с помощью клавиши Tab

  • установка отборов, при которых будет показываться только номенклатура с установленной ценой или запасы, имеющиеся в наличии

  • Обновлять заголовки. Доступно при включении использования характеристик или нескольких складов и позволяет включить/выключить обновление заголовков соответствующих списков (характеристик номенклатуры и складские остатки)

  • Сведения о документе. Открывает форму с данными о документе, из которого произведен вызов подбора

     

    Дополнительная информации о подборе

    Подбор позволяет работать либо с полнотекстовым, либо контекстный поиском. Выбор варианта поиска регулируется глобальной настройкой. Таким образом, если в программе используется полнотекстовый поиск в подборе есть возможность его тонкой настройки. Опции тонкой настройки расположены в меню "Еще", подменю "Поиск" и являются взаимоисключающими.

    Опция: поиск слов с любыми окончаниями

    При поиске слов с включенной опцией результат будет содержать слова с любыми окончаниями. Пример: если ввести "доку" – поиск найдет слова "документ", "документировать", "документальный" и др. Опция по умолчанию включена.

    Опция: поиск слов с опечатками

    Если опция включена допускается отличие искомого слова на один, любой, символ. При этом слово должно быть введено целиком. Пример: запрос "система" найдет "систама", "сивтема". Опция по умолчанию выключена.

    Помимо полнотекстового/контекстного поиска в списках есть возможность использовать типовые возможности поиска.

    ВАЖНО! Вызов поиска по списку осуществляется командой CTRL+F либо началом набора искомого текста.

    Необходимо понимать, что полнотекстовый\контекстный поиск и поиск по списку – это разные инструменты, и результат их совместного использования будет складываться. Так, например, полнотекстовым поиском можно отобрать все позиции номенклатуры, в наименовании которых содержится строка "крепеж" (на любой позиции, не обязательно в начале), а после поиском по списку выбрать номенклатуру, наименование которой начинается со слова "строительный".

    Отмена поиска:

  • для полнотекстового поиска необходимо очистить поле с введенным текстом (в шапке подбора);

  • для поиска по списку необходимо активизировать нужный список и нажать CTRL+Q;

    ВАЖНО! Включение полнотекстового поиска выполняется в настройках приложения.

    Подбор товаров может быть произведен с запросом количества и цены или без запроса количества и цен (кнопка Настройка):

  • при установке флажков будет производиться запрос заданного параметра позиции номенклатуры

  • если ни один из флажков не установлен, то при повторном выборе позиции номенклатуры количество (и резерв) будет увеличиваться на 1

    Выбранные позиции номенклатуры автоматически переносятся в табличную часть документа.

    Для переноса подобранных позиций в документ используется кнопка Перенести в документ.

    ВАЖНО! Подбор не используется для редактирования заполненных табличных частей.
