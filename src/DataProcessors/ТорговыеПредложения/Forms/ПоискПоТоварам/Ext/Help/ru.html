Поиск по товарам

В данной форме производится поиск торговых предложений по списку товаров, при этом подбираются самые оптимальные предложения поставщиков, которые обладают максимальным охватом искомых позиций с минимальными ценами и сроками поставки.

Регион поиска

Возможно указать отбор по регионам, указанным в настройках профиля абонента или региону выбранному вручную.

Начало работы с поиском

Заполнение списка товаров производится на основании документов, например, Заказы поставщикам.

Для заполнения из формы документа через меню Создать на основании необходимо выполнить команду Заказы поставщикам в 1С:Бизнес-сеть.

Настройка поиска

Параметры поиска настраиваются по команде Настроить поиск:

  • Превышение минимальной цены, % (не более) — максимальное отклонение от минимальной цены, при котором цена будет считаться оптимальной.
  • Срок поставки, дней (не более) — максимальный срок поставки, при котором срок будет считаться оптимальным.
  • По наименованию (по умолчанию) — торговые предложения будут искаться по полному или частичному совпадению наименования искомой номенклатуры.
  • По характеристике (не может быть выбрано без поиска по наименованию) — торговые предложения будут искаться по полному или частичному совпадению названия характеристики искомой номенклатуры.
  • По артикулу — торговые предложения будут искаться по полному совпадению артикула искомой номенклатуры.
  • По штрихкоду — торговые предложения будут искаться по полному совпадению штрихкодов искомой номенклатуры.

Работа с результатами поиска

Выбор найденных предложений

Для выбора предложения из списка результатов нужно установить флаг напротив его цены.

Автовыбор

Существует возможность автовыбора торговых предложений по двум критериям:

  • По минимальной цене — будут выбраны торговые предложения с минимальной ценой.
  • По минимальным срокам — будут выбраны торговые предложения с минимальным сроком поставки.

В качестве результата поиска для каждого искомого товара будут показаны найденные торговые предложения, которые может предоставить поставщик. Поставщики будут отсортированы по мере убывания оптимальности их предложения в целом для всего списка искомых товаров. При оценке оптимальности учитывается общий охват поставщиком искомых позиций, количество оптимальных цен и сроков поставки.

Если не найдены поставщики по товару, отображается гиперссылка приглашения собственных поставщиков в сервис 1С:Бизнес-сеть.

Запросы цен

Оформление запросов цен на товары поставщиков по товарам, добавленным в корзину, производится по команде Запросить цены. При выполнении команды открывается форма для подготовки запросов цен и отправки запросов поставщикам.

Оформление заказов

После того как торговые предложения будут выбраны, они автоматически добавляются в корзину.

Для оформления заказов необходимо выполнить команду Оформить заказы или нажать на гиперссылку в правом нижнем углу, в которой указана информация о подобранных товарах в корзине. При выполнении команды открывается форма Оформления заказов, где производится оформление и отправка документов поставщикам.
