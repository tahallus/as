Регион поиска

Возможно указать отбор по регионам, указанным в настройках профиля абонента, или региону, выбранному вручную.

Навигация по категориям

Реализована навигация по категориям рубрикатора:

  • в панели навигации с иерархическим списком категорий;
  • в выпадающем списке при вводе первых символов наименования в поисковой строке. Например, при вводе строки теле появится список выбора со значением Телевизоры.

Отборы

Возможно использование дополнительных отборов для фильтрации списка торговых предложений по доступным характеристикам выбранной категории товаров.

Для включения отборов необходимо выбрать категорию товаров и нажать кнопку панели Показать отборы. Справа отобразится панель, содержащая общие отборы (поставщик, цена) и характеристики товаров категории.
При установке значений отбора повторный поиск производится автоматически.

Список результатов

В списке результатов отображаются найденные торговые предложения, соответствующие параметрам поиска.

Сортировка результатов

Для результатов поиска возможна сортировка по наименованию и по цене (по возрастанию и убыванию).

Добавление в корзину

Для добавления торговых предложений в корзину необходимо нажать на надпись Добавить в колонке Корзина. В зависимости от режима ввода количество будет увеличиваться на 1 либо откроется окно ввода количества.

Установка режима ввода количества производится по команде Запрашивать количество в меню Еще.

  • При снятом флаге (по умолчанию) при нажатии Добавить количество увеличивается на единицу.
  • При установленном флаге при нажатии Добавить количество будет запрашиваться в диалоговом окне.

Запросы цен

Оформление запросов цен на товары поставщиков по товарам, добавленным в корзину, производится по команде Запросить цены. При выполнении команды открывается форма для подготовки запросов цен и отправки запросов поставщикам.

Оформление заказов

Оформление заказов по товарам, добавленным в корзину, производится по команде Оформить заказы. При выполнении команды открывается форма Оформления заказов, где производится оформление и отправка заказов поставщикам.
