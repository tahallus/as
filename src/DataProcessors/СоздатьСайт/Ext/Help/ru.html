Создание сайта

Из 1С:УНФ можно создать сайт в один клик, заполнить его товарами и рассказать о нем друзьям и клиентам. 

Для этого используется облачный сервис готовых сайтов UMI.ru.

При создании сайта из 1С заполняется контактная информация (имя организации, адрес, телефон, e-mail), обменом из 1С на сайт переносятся товары, цены, картинки товаров, а созданные на сайте заказы передаются в 1С как документы «Заказ покупателя».


  1. Выберите тип сайта: интернет-магазин, сайт компании, лендинг или сайт специалиста.

  2. Проверьте e-mail, на который будет зарегистрирован сайт, и имя сайта.

  3. При необходимости, выберите в дополнительных настройках организацию, из которой будет взята контактная информация для сайта, для интернет-магазина дополнительно можно указать вид цен, режим отображения остатков, категории / группы товаров, которые будут перенесены на сайт.


Нажмите «Создать сайт» и подождите несколько минут, сайт откроется в окне браузера.


Для редактирования сайта нужно зайти в админзону, это можно сделать через меню Компания – Сайт компании – Открыть сайт. Выберите в списке «Открыть админзону».


У 1C-UMI.ru существует более 550 шаблонов для оформления сайтов, выберите подходящий, перейдя в админзоне сайта в меню Дизайн – Смена дизайна.


Онлайн-запись на услугу в 1С

Чтобы записываться на услугу на сайте, вам нужно ввести в 1С список мастеров (ресурсов), задать для них график работы, ввести услуги и их цены.

Проверьте настройки расписания:

1. Создайте графики работы в справочнике Работы - Графики работы.

В графике можно использовать шаблоны расписания.


2. Введите мастеров (ресурсы) в справочнике Работы - Ресурсы.
Ресурсом может быть сотрудник (парикмахер) в салоне красоты, столик в кафе и др. К ресурсу будет привязано время записи (бронирования).
В ресурсе установите "Шаг планирования" - длительность услуги, "Контролировать шаг планирования", "Загрузка только в рабочее время".
Мастера могут смотреть данные о записи клиентов в личном графике Google, если в календаре сотрудника установлен флажок "Синхронизировать с Google"

3. В Ресурсе перейдите по навигационной ссылке "График работы" и нажмите "Установить график с выделенной даты".


4. Для работы с записью на услуги можно использовать Планировщик ресурсов. В панели справа установите флажок “По графику работы” и “Заказ-наряды”, задайте “Интервал шкалы” как интервал записи для услуг.

Проверьте настройки для услуг:


1. Создайте услуги, на которые можно будет записаться на сайте: Продажи - Номенклатура


2. Установите для них цены: Продажи - Формирование цен, или в карточке товара по ссылке “Цены”

После ввода настроек, откройте заново форму создания сайта, выберите мастеров и услуги, которые они оказывают, флажками в списке
