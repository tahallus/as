Обработка "Прайс-лист контрагентов (поставщиков)"

Обработка предназначена для работы с входящими ценами (цены поставщиков).

 

  • Внешний вид;
  • Цены;
  • Колонки;
  • Фильтры;
  • Оформление;
  • Дополнительная информация;

 

Внешний вид

- Рабочий прайс-лист. Позволяет быстро выбрать (переключить) прайс-лист поставщиков для работы. Поле дополнено командами "Создать" и "Скопировать".

- Иерархия. Опция позволяет выбрать, по какому справочнику построить дерево прайс-листа. Кроме классического варианта, с иерархией по папкам номенклатуры, присутствует возможность построить прайс-лист по элементам ценовых групп или категорий номенклатуры. Т.е. справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться. Примером использования, помимо прочих, могут быть случаи, когда папки номенклатуры заводились по производителям, а прайс-лист необходимо построить по продукции и фактически объединить всех производителей в одну группу прайс-листа.

 

Цены

Группа содержит настройку Периода (дата), на который будут получены цены. Если значение не указано, будут выбраны актуальные данные.

Если в программе включено использование нескольких валют группа будет дополнена реквизитами Валюта и дата курса. Валюта - цены во всех колонках будут пересчитаны по курсу на указанную дату в текущую валюту.

 

Колонки

Группа содержит настройки колонок, которые будут выведены в прайс-лист:

  • Колонки видов цен поставщиков. Какие цены будут выведены в документ ("Цена поставщика 1", "Цена поставщика 2" и т.д.).

 

Фильтры

Позволяет настроить универсальные отборы (по любому реквизиту номенклатуры, в т.ч. папкам, ценовым группам или категориям). Задать настройку выводить только те позиции, которые отсутствуют на складах или номенклатуру без цен. Так же в данной группе можно установить опцию сравнения цен (в документе будут отображаться строки по условию сравнения цен).

 

Оформление

В группе собраны настройки флажков, которые указывают, какие ячейки табличного документа будут выделены цветом (промаркированы). Например, пользователь может указать, что необходимо выделить минимальную цену. Для прайс-листа может быть выбран один тип маркировки. Маркировки Минимум и Максимум доступны, если выбрано несколько видов цен поставщиков. 

 

Дополнительная информация

В данной закладке хранятся настройки вывода даты формирования, наименование прайс-листа поставщиков. Еще данная закладка хранит текст подвала прайс-листа, который может содержать дополнительную текстовую информацию, например, краткое памятку об условиях закупки товаров.

Нужно еще отметить, что в обработке присутствует опция, которая указывает, что необходимо открывать пользователю по команде Прайс-листы поставщиков: список сохраненных прайс-листов или непосредственно обработку формирования.

Список всех прайс-листов поставщиков находится в меню Все справочники раздела Компания.

 

ВАЖНО! Следует понимать, что настройки хранит справочник, а обработка оперирует опциями при построении прайс-листа поставщиков.
