Для редактирования персональных настроек можно воспользоваться командой Мои настройки в разделе Настройки.

Работа с файлами

На вкладке производятся персональные настройки для работы с файлами:
  • Действие при выборе файла - укажите действие программы при выборе файла в списке с помощью нажатия на клавишу Enter или двойном щелчке мыши на файле - открыть файл или открыть его карточку.
  • Выбор режима открытия при выборе файла - включите флажок, для того чтобы программа предлагала выбрать режим открытия файла при его выборе. Если флажок включен, то перед открытием файла откроется окно для выбора режима открытия - просмотр или редактирование.
  • Показывать подсказки при редактировании файлов (только веб-клиент) - включить подсказки при редактировании файлов при работе в веб-клиенте. Если флажок включен, то при работе через веб-клиент программа будет подсказывать, как удобнее открыть файл для просмотра или редактирования, как обновить его в информационной базе.
  • Показывать занятые файлы при завершении работы - если флажок включен и текущий пользователь занял один или несколько файлов, то при завершении работы показывается список этих файлов. Можно открыть файл для просмотра, закончить редактирование, открыть карточку файла.
  • Показывать колонку "Размер" в списках файлов - если флажок включен, в списке файлов и в списке редактируемых файлов показывается колонка "Размер" (в Кб).
  • Показывать информацию, что файл не был изменен - если флажок включен, то при сохранении версий файла программа будет сообщать, что в файл не было внесено изменений. При выполнении команды Закончить редактирование или Сохранить изменения, если файл не был изменен, новая версия не создается (существующая не изменяется), если включен флажок, то выходит сообщение, что файл не менялся.
  • Сравнивать версии файлов ODT, DOC, RTF, TXT при помощи - выбор одного из двух возможных вариантов сравнения файлов указанного формата - Microsoft Office Word или OpenOffice.org Writer.
  • Настройки открытия файлов - в предлагаемой таблице выберите способ открытия для текстовых файлов с указанными расширениями (во встроенном редакторе программы или стандартным приложением, например, текстовый файл вместо встроенного редактора программы можно открыть с помощью Блокнота Windows).
  • Настройка основного рабочего каталога - перейдите по ссылке, для того чтобы указать параметры основного рабочего каталога.
  • Настройка сканирования - переход к настройке параметров сканирования.
