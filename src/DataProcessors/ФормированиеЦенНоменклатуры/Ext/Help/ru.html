Обработка "Формирование цен"

Обработка предназначена для пакетного расчета, изменения или удаления цен номенклатуры.

 

Помощник содержит следующие шаги:

- выбор видов цен (колонок прайс-листа), по которым предполагается сформировать новые цены;

- настройка видимости колонок и дополнительных опций;

- страница формирования новых цен;

- опции записи цен;

- страница с итогами работы помощника.

 

Для страницы формирования новых цен определены две группы команд. Подобрать номенклатуру:

  • по приходным накладным – добавить номенклатуру из приходной накладной
  • по номенклатурной группе – добавить всю номенклатуру, которая относится к определенной номенклатурной группе
  • по ценовой группе - добавить всю номенклатуру, которая относится к определенной ценовой группе
  • по виду цен – команда позволяет добавить номенклатуру, для которой не заполнены цены, или у которой есть записи по виду цен (колонкам прайс-листа)
  • из списка номенклатуры – добавить произвольную номенклатуру из общего списка номенклатуры (реализовано две возможности: добавление по элементам, добавление по группам)

 

Команды группы Действия:

  • Изменить на сумму
  • Изменить на процент
  • Округление
  • Очистить
  • Изменить по формуле

 

* - Следует понимать, что для копирования записей из одного вида цен (колонки прайс-листа) в другой необходимо выполнить два действия:

  1. заполнить состав номенклатуры по исходному виду цен (группа команд "Подобрать номенклатуру")
  2. заполнить цены для номенклатуры, например, при помощи команды "Действия - Изменить по формуле"
