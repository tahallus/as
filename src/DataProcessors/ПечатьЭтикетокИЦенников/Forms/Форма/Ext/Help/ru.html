Печать этикеток и ценников

Сервис Печать этикеток и ценников предназначен для формирования списка номенклатуры, для которой нужно распечатать этикетки и ценники, и, собственно, для их печати по заранее созданным шаблонам.

Шаблоны можно создать в списке, который открывается по ссылке Настройка и администрирование - Печатные формы, отчеты и обработки - Шаблоны этикеток, ценников и чеков ККМ.

Сервис Печать этикеток и ценников можно запустить по одноименной ссылке из подраздела Сервис панели действий раздела Продажи, а также из меню Печать документов Поступление товаров, Перемещение товаров.

Изменение режима печати

В меню Установить режим печати командной панели формы можно выбрать следующие режимы:

  • печать ценников,
  • печать ценников и этикеток,
  • печать этикеток.


В зависимости от выбранного режима изменяется состав колонок в табличной части.

Заполнение основных реквизитов

Дата и вид цены для формирования этиекток и ценников указывается в шапке сервиса. Там же можно указать вид цены по акции.
Если вид цены не указан, используется вид цены продажи пользователя по умолчанию.

Заполнение табличной части

Если сервис Печать этикеток и ценников был запущен из документов, то табличная часть автоматически заполняется номенклатурой исходных документов.

Если сервис был запущен вручную, то можно настроить отбор товаров, нажав кнопку Показать отбор в командной панели табличной части. После настройки параметров отбора можно заполнить табличную часть по ссылке Заполнить по отбору из меню Заполнить. Также можно заполнить табличную часть с помощью стандартного сервиса Подбор товаров, который запускается по кнопке Подбор.

Редактирование табличной части

Выделять строки табличной части можно стандартными способами с помощью мыши и комбинаций клавиш на клавиатуре.
Для выделенных строк из меню Установить для выделенных строк можно выбрать следующие пункты:

  • Количество ценников - вводится фиксированное значение;
  • Шаблон ценников - выбирается из списка всех шаблонов с типом Этикетка (ценник);
  • Количество этикеток - вводится фиксированное значение;
  • Шаблон этикеток - выбирается из списка всех шаблонов с типом Этикетка (ценник);
  • Количество по умолчанию - устанавливается количество по умолчанию, для ценников - 1, для этикеток - количество, равное остатоку на складе или количеству из исходного документа, в зависимости от того, откуда был запущен сервис.


Можно выбрать шаблон или изменить количество для конкретной позиции в соответствующей ячейке табличной части.

В первой колонке табличной части отмечаются позиции, для которых будет выполнена печать. Все поля с шаблонами и количествами в этих позициях должны быть заполнены. Отмечать позиции можно вручную и с помощью сервисных кнопок командной панели.

Печать этикеток и ценников

По нажатию кнопки Печать в командной панели сервисной формы для отмеченных позиций генерируются печатные формы ценников и этикеток. Перед отправкой на принтер при необходимости пользователь может внести исправления в созданные печатные формы.
