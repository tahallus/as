Обработка "Загрузка данных из Бухгалтерии предприятия 3.0"

С помощью этой обработки можно загрузить данные из конфигурации Управление торговлей, редакция 10.3. В форме обработки следует указать файл обмена, в который были выгружены данные из конфигурации Управление торговлей, редакция 10.3, и нажать на кнопку Загрузить.

Важно. Перенос данных из конфигурации Бухгалтерия предприятия, редакция 3.0 возможен, начиная с версии 3.0.87.

Файлы, необходимые для выгрузки данных из конфигурации Бухгалтерия предприятия, редакция 3.0 входят в поставку программы Управление фирмой, редакция 1.6 и находятся в каталоге ConvAccounting3.

  • Перенос данных БП 3 - УНФ 1.6.epf -внешняя обработка выгрузки данных во внешний файл из конфигурации Бухгалтерия предприятия, редакция 3.0.
  • ConvAccounting3.txt - инструкция по переходу с описанием порядка и особенностей переноса данных объектов.

Порядок действий при переносе данных

Выполните резервное копирование информационной базы конфигурации "Бухгалтерия предприятия" редакции 3.0. Если версия конфигурации "Бухгалтерия предприятия" ниже 3.0.87, то проведите обновление конфигурации до указанной или более поздней версии. Выполните все регламентные процедуры в конфигурации "Бухгалтерия предприятия", редакция 3.0.

  • Удалите помеченные на удаление объекты.
  • Закройте кассовые смены.
  • Выполните регламентные операции.

Важно. На момент переноса данных в информационной базе "Бухгалтерия предприятия" редакции 3.0 не должно быть отрицательных остатков товаров по складам и организациям. Информация об отрицательных остатках товаров не будет выгружена из информационной базы конфигурации.
  • Откройте внешнюю обработку "Перенос данных БП 3 - УНФ 1.6.epf" с помощью пункта меню "Файл"-"Открыть".
  • В появившемся диалоге укажите имя внешнего файла, в который будет производиться выгрузка данных.
  • В диалоговом окне появится список объектов, которые могут быть выгружены. По умолчанию будет выгружена информация по справочникам и остаткам, рекомендуемым к выгрузке. При необходимости можно отключить выгрузку информации по любому из справочников или отключить выгрузку остатков.
  • При выгрузке остатков задолженности клиентов будет выгружен документ "Ввод остатков" с соответствующим видом операции, с указанием суммы непогашенной задолженности.
  • После настройки правил выгрузки данных необходимо нажать на кнопку "Выгрузить данные". Данные будут выгружены во внешний файл.
  • Произведите установку конфигурации Управление фирмой, редакция 1.6.
  • Создайте новую информационную базу конфигурации Управление фирмой, редакция 1.6.
  • В конфигурации "Управление фирмой" сопоставьте предопределенную организацию "Наша фирма" с соответствующей организацией конфигурации Бухгалтерия предприятия. Для этого добейтесь соответствия реквизитов организации "ИНН" и "Наименование" в обоих информационных базах.
  • Загрузите курсы валют и включите опции учета (несколько организаций, комиссионную торговлю и т.д.).
  • Для загрузки данных выберите раздел "Компания" - "Загрузка данных из БП 3.0". В открывшемся диалоговом окне укажите внешний файл, в который была произведена выгрузка, и нажмите на кнопку "Загрузить". Данные будут загружены в информационную базу из внешнего файла.
  • После проведения загрузки убедитесь, что данные загружены корректно. Остатки товаров, денежных средств и взаиморасчетов фиксируются документом "Ввод остатков". С помощью отчетов проверьте правильность выгрузки остатков в обеих базах.
Важно. Функционал предназначен для разового перехода, при повторных загрузках корректность данных не гарантируется.
