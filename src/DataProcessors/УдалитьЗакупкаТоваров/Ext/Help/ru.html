Обработка "Закупка товаров"

Данная обработка предназначена для расчета потребности товаров на основании статистики продаж за прошлый период и автоматического формирования заказов поставщикам.

Заполнение табличной части Товары и расчет потребности производится по кнопке Рассчитать потребность. Предварительно необходимо указать количество дней, на которое должна быть рассчитана потребность и период для расчета статистики продаж. Дополнительно существует возможность ограничить состав товаров только товарами определенного поставщика или товарами определенной группы номенклатуры.

Табличная часть Товары представлена в виде дерева, строки которого сгруппированы по поставщикам:

  • Поставщик - поставщик, указанный в карточке номенклатуры на закладке Параметры хранения и закупки;
  • Номенклатура - закупаемый товар;
  • Характеристика - характеристика товара;
  • Текущий остаток - текущий остаток товара на складах предприятия;
  • Продано (кол-во) - проданное количество товара за период расчета статистики;
  • Продавалось (дней) - количество дней в периоде расчета статистики, в которые товар был на остатках;
  • В среднем за день - среднее количество продаж за день, рассчитываемое по формуле: Продано (кол-во) / Продавалось (дней);
  • Рекомендуется закупить - потребность в товаре, рекомендуемая к закупке. Рассчитывается по формуле: В среднем за день * Количество дней - Текущий остаток, где Количество дней - количество дней на которое должна быть рассчитана потребность. При расчете дробные значения округляются в большую сторону. Значение в данной колонке может быть изменено пользователем.

Заполнение табличной части Товары производится в соответствии с параметрами, установленными в шапке обработки. По умолчанию отображаются только товары, по которым были продажи за период расчета статистики. Для отображения всех товаров, в том числе и тех, по которым не было продаж, необходимо нажать на кнопку Показать нулевые продажи.

Формирование заказов поставщикам производится по кнопке Сформировать заказы. При этом по каждому поставщику для отмеченных строк создается отдельный заказ. Колонка Количество табличной части заказа заполняется значением колонки Рекомендуется закупить. Реквизиты шапки заказа заполняются значениями по умолчанию, установленными для текущего пользователя. Цены закупки в табличной части заказа заполняются по виду цен, указанному в договоре по умолчанию контрагента.

Результат формирования заказов отображается на закладке Заказы. После формирования необходимо ознакомится с результатом заполнения реквизитов документов и заполнить в заказах дату поступления товара. Для отмеченных строк можно выполнить групповое проведение или удаление заказов.
