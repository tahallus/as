Обработка "Прайс-лист организации"

Обработка предназначена для просмотра и редактирования отпускных цен нашей компании.

 

  • Внешний вид;
  • Цены;
  • Колонки;
  • Фильтры;
  • Оформление;
  • Дополнительная информация;

 

Внешний вид

- Рабочий прайс-лист. Позволяет быстро выбрать (переключить) прайс-лист для работы. Поле дополнено командами "Создать" и "Скопировать".

- Вид прайс-листа. Предоставляется возможность выбора вида компоновки: полотно, две колонки, диафильм.

  • Полотно – является классическим вариантом компоновки данных, при котором номенклатура, ее описание, цены и другая информация располагаются в строку, а сам прайс-лист выглядит в форме одной монолитной таблицы.
  • Две колонки – представляет собой способ подачи информации с максимальной компактностью. В связи с этим имеет некоторые ограничения, например, выводится только одна колонка с ценой, отсутствует возможность вывода картинки и других второстепенных данных.
  • Диафильм (слайды) – по внешнему виду напоминает каталог товаров. Основное назначение – предоставить пользователю максимально наглядный прайс-лист, где номенклатура представлена в виде картинок.

- Иерархия. Опция позволяет выбрать, по какому справочнику построить дерево прайс-листа. Кроме классического варианта, с иерархией по папкам номенклатуры, присутствует возможность построить прайс-лист по элементам ценовых групп или категорий номенклатуры. Т.е. справочник номенклатуры может содержать один набор папок, а иерархия прайс-листа может отличаться. Примером использования, помимо прочих, могут быть случаи, когда папки номенклатуры заводились по производителям, а прайс-лист необходимо построить по продукции и фактически объединить всех производителей в одну группу прайс-листа.

 

Цены

Группа содержит настройку Периода (дата), на который будут получены цены. Если значение не указано, будут выбраны актуальные данные.

Если в программе включено использование нескольких валют группа будет дополнена реквизитами Валюта и дата курса. Валюта - цены во всех колонках будут пересчитаны по курсу на указанную дату в текущую валюту.

 

Колонки

Группа содержит настройки колонок, которые будут выведены в прайс-лист. Настройки разделены на два типа:

  • Состав реквизитов номенклатуры. Какие данные номенклатуры необходимо указать в прайс-листе (наименование, картинка, страна происхождения, новинка и т.д.).
  • Колонки видов цен. Какие цены будут выведены в документ (оптовая, розница и т.д.).

Следует отметить, что сам признак новинки хранится в номенклатуре, равно как и указания количественного признака границы для остатков.

 

Фильтры

Позволяет настроить универсальные отборы (по любому реквизиту номенклатуры, в т.ч. папкам, ценовым группам или категориям). Задать настройку выводить только те позиции, которые есть в наличии или номенклатуру без цен. Так же в данной группе можно установить опцию сравнения цен (в документе будут отображаться строки по условию сравнения цен).

 

Оформление

В группе собраны настройки флажков, которые указывают, какие ячейки табличного документа будут выделены цветом (промаркированы). Например, пользователь может указать, что необходимо выделить минимальную цену. Для прайс-листа может быть выбран один тип маркировки. Маркировки Минимум и Максимум доступны, если выбрано несколько видов цен. 

Выделение цветом по условию не используется в прайс-листах с вариантом печати "Диафильм (слайды)".

 

Дополнительная информация

В данной закладке хранятся настройки вывода контактной информации, подписи даты формирования и представления остатков. Еще данная закладка хранит текст подвала прайс-листа, который может содержать дополнительную текстовую информацию, например, краткое описание заключения сделок.

Нужно еще отметить, что в обработке присутствует опция, которая указывает, что необходимо открывать пользователю по команде Прайс-листы организации: список сохраненных прайс-листов или непосредственно обработку формирования.

Список всех прайс-листов организации находится в меню Все справочники раздела Компания.

 

ВАЖНО! Следует понимать, что настройки хранит справочник, а обработка оперирует опциями при построении прайс-листа.
