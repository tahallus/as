Отчет "Финансовый результат"

Отчет представлен в двух вариантах:

  • Финансовый результат (прогноз)
  • Финансовый результат (план-фактный анализ)

Финансовый результат (прогноз)


В отчет выводятся сведения о прогнозе финансового результата на заданный период для выбранной организации.

Отчет формируется на основании данных документов Бюджет.

Финансовый результат (план-фактный анализ)


В отчет выводятся сведения о сравнительном анализе плановых и фактических данных финансового результата на заданный период для выбранной организации.

Плановые данные формируются на основании данных документов Бюджет.
