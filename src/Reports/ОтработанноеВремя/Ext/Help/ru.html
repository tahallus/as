Отчет "Табель"

В отчете отображаются сводные данные по учету рабочего времени на основании документов Табель. Отчет может быть сформирован в двух режимах: По дням и В целом.

В режиме По дням в отчете отображаются сведения о ежедневных затратах рабочего времени за указанный период. Данные группируются по подразделениям.


В режиме В целом за период в отчет выводятся данные о суммарных затратах рабочего времени с детализацией до вида рабочего времени за указанный период. Данные группируются по сотрудникам и подразделениям.
