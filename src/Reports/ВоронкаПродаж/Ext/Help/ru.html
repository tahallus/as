Отчет "Воронка продаж"

Воронка продаж - один из самых эффективных инструментов CRM-системы. Отчет наглядно показывает количественное соотношение лидов, покупателей и заказов на разных этапах. В ней отражен полный цикл продажи, начиная с заявки клиента и заканчивая завершением заказа. Состав и порядок этапов в воронке продаж соответствуют списку состояний в заказах.

Колонки отчета:

  • Состояние - состояние заказа, как оно настроено в справочнике Вид заказа.
  • Кол-во - количество заказов, прошедших через текущее состояние или находящихся в нём. Заказы, перепрыгнувшие через состояние, считаются побывавшими в этом состоянии. Шкала отображает соотношение к заказам, побывавшим в самом первом состоянии.
  • Конверсия - процент заказов перешедших из предыдущего состояния.
  • t, ч - среднее время, проведенное заказами в текущем состоянии.
  • Сумма - сумма заказов, прошедших через текущее состояние.
  • Потери - количество отмененных в текущем состоянии заказов . Шкала пропорционально показывает распределение отмен по причинам. Процент потерь рассчитывается от количества заказов побывавших в текущем состоянии.

 

Отчет представлен в следующих вариантах:

  • Воронка продаж
  • Воронка продаж по лидам 
  • Воронка продаж по покупателям
  • Воронка продаж по заказам

 

Воронка продаж


Вариант Воронка продаж показывает общую воронку по всем лидам, покупателям и заказам. Вариант поможет увидеть в одном месте информацию по всем главным объектам CRM системы.

Фильтры отчета:
В отчете можно настроить фильтры по периоду и по менеджеру. При выборе фильтров в варианте воронки продаж будут отображены лиды, покупатели и заказы, удовлетворяющие заданным фильтрам.

Сравнение воронки:
  • По менеджерам. Сравнение воронки продаж по менеджерам позволит проанализировать работу с заказами по отдельно взятым менеджерам. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных менеджеров, по которым нужно построить сравнение.

 

Воронка продаж по лидам


Вариант воронки продаж по лидам позволяет увидеть распределение лидов по стадиям процесса продаж, а также увидеть какое количество лидов было сконвертировано в покупателей и сколько заказов в последствии сделали данные покупатели. Кроме того в отчете можно будет увидеть на какую сумму были сделаны заказы покупателей, сконвертированные из лидов. Таким образом, можно будет проследить весь путь работы с лидом, начиная от его создания и заканчивая завершением работы с заказом, совершенным данным лидом. Вариант отчета поможет определить какую сумму продаж в конечном итоге приносят лиды.

Фильтры отчета: В отчете можно настроить фильтры по периоду и по всем реквизитам лидов, покупателей и заказов.
Важно! При выборе фильтров в варианте воронки продаж по лидам, будут отображены лиды, удовлетворяющие заданным фильтрам и показаны все заказы, совершенные отобранными по фильтрам лидами.

Сравнение воронки:

  • По лидам. Сравнение воронки продаж поможет увидеть воронки по отдельно взятым лидам. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных лидов, по которым нужно построить сравнение.
  • По менеджерам. Сравнение воронки продаж по менеджерам позволит проанализировать работу с лидами по отдельно взятым менеджерам. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных менеджеров, по которым нужно построить сравнение.

 

Воронка продаж по покупателям


Вариант воронки продаж по покупателям позволяет увидеть конверсию покупателей в заказы, а также увидеть какой процент покупателей в итоге завершает заказ успешно, а так же совершает повторный заказ. Данные показатели помогут понять лояльность покупателей и определить насколько хорошо менеджеры работают с уже существующими клиентами.

Фильтры отчета: В отчете можно настроить фильтры по периоду и по всем реквизитам лидов, покупателей и заказов.
Важно! При выборе фильтров в варианте воронки продаж по покупателям, будут отображены покупатели, удовлетворяющие заданным фильтрам и показаны заказы, совершенные отобранными по фильтрам покупателями.

Сравнение воронки:

  • По покупателям. Сравнение воронки продаж поможет увидеть воронки по отдельно взятым покупателям. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных покупателей, по которым нужно построить сравнение.
  • По менеджерам. Сравнение воронки продаж по менеджерам позволит проанализировать работу с покупателями по отдельно взятым менеджерам. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных менеджеров, по которым нужно построить сравнение.

 

Воронка продаж по заказам


Вариант воронки продаж по заказам наглядно отображает распределение заказов по стадиям процесса продаж. Формируется на основании истории изменения состояний заказов покупателей и заказ-нарядов.

Фильтры отчета: В отчете можно настроить фильтры по периоду и по всем реквизитам лидов, покупателей и заказов. При выборе фильтров в варианте воронки продаж по заказам, будут отображены заказы, удовлетворяющие заданным фильтрам.

Сравнение воронки:

  • По покупателям. Сравнение воронки продаж поможет увидеть воронки по отдельно взятым покупателям. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных покупателей, по которым нужно построить сравнение.
  • По менеджерам. Сравнение воронки продаж по менеджерам позволит проанализировать работу с заказами по отдельно взятым менеджерам. Для формирования воронки необходимо выбрать несколько групп или же несколько конкретных менеджеров, по которым нужно построить сравнение.
  • По подразделениям: отчет формируется с группировкой по подразделению. Для формирования требуется задать фильтр по подразделениям.
