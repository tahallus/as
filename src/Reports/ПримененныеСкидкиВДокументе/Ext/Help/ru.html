Отчет "Примененные скидки в документе"

Отчет предназначен для проведения анализа примененных автоматических скидок (наценок) по соответствующему документу.
Отчет доступен при включенной функциональной опции Использовать автоматические скидки и наценки (раздел Настройки - Продажи).

Особенности отчета:

  • Группировка всех автоматически применяемых скидок (наценок) по условиям их применения;
  • Анализ всех не примененных скидок (наценок) по причине не выполнения условий по их применению;
