Отчет "Оборотно-сальдовая ведомость"

Отчет содержит информацию для каждого счета об остатках на начало и конец периода и оборотах по дебету и кредиту за период.

 

Ведомость может быть построена по данным учета выбранной организации.

По кнопке «Сформировать» выполняется построение отчета в соответствии с текущей настройкой.

 

Реквизиты отчета:

  • Период. Даты начала и конца периода отчета.
  • Организация. Организация, по данным учета которой строится ведомость.
  • Счет. Счет, по которому формируется ведомость. Если не указан, то по всем счетам.
  • Валютная сумма. Если флажок установлен, для валютных счетов будут выводиться валютные суммы.
