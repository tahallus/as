Отчет "Воронка по лидам"

Наглядно отображает распределение лидов по стадиям процесса продаж. Формируется на основании истории изменения состояний лидов.

Колонки отчета:

  • Состояние - состояние лида, как оно настроено в справочнике "Состояния лидов".
  • Кол-во - количество лидов, прошедших через текущее состояние или находящихся в нём. Лиды, перепрыгнувшие через состояние, считаются побывавшими в этом состоянии. Шкала отображает соотношение к лидам, побывавшим в самом первом состоянии.
  • Конверсия - процент лидов перешедших из предыдущего состояния.
  • t, ч - среднее время, проведенное лидами в текущем состоянии.
  • Потери - количество некачественных лидов в текущем состоянии. Шкала пропорционально показывает распределение некачественных лидов по причинам. Процент потерь рассчитывается от количества лидов побывавших в текущем состоянии.

 

Варианты отчета:

  • По лидам - отчет формируется по всем лидам.
  • По менеджерам - отчет формируется с группировкой по ответственному за работу с лидом. Для формирования требуется задать фильтр по сотрудникам или группам сотрудников.
  • По источникам привлечения- отчет формируется с группировкой по источникам привлечения. Для формирования требуется задать фильтр по источникам привлечения.
