Отчет "Анализ базы лидов"

 

Предоставляет информацию о динамике развития базы лидов и ее распределению по ответственным менеджерам.

График отчета “Анализ базы контрагентов” покажет динамику создания новых лидов по дням, неделям, месяцам.

Нижняя круговая диаграмма показывает разбивку потенциальных клиентов по менеджерам. Цветовая легенда показывает каким цветом окрашивается область каждого менеджера.

Анализ можно проводить не по всей базе, а по частям используя фильтры в верхней панели отчета. Фильтровать базу можно по тегам, состояниям и истояникам привлечения лидов.
