Строка поиска

  • С помощью строки поиска в форме отчета можно установить отбор по найденному вхождению (название товара, имя покупателя).
  • Автоподбор значения в строке поиска выполняется по полю, добавленному в раздел Фильтры.

Печать отчета

  • Сформированный отчет можно распечатать, нажав кнопку Печать.

Отправка отчета по электронной почте

Отправка отчета по электронной почте осуществляется по кнопке Отправить, для этого:

  • с помощью флажков выберите один или несколько форматов, в которых необходимо отправить отчет.
  • при необходимости включите флажок Дополнительно упаковать в архив (.zip).
  • Нажмите Продолжить. Отчет будет прикреплен к электронному письму в качестве вложения.

Сохранение результата отчета в файл

  • Для сохранения результата отчета в файл нажмите Сохранить, укажите наименование файла и путь для сохранения.

Раскрыть на весь экран 

  • Чтобы отобразить отчет в окне для просмотра, без управляющих элементов, нажмите кнопку .

Варианты отчета

  • В программе может быть предусмотрено несколько вариантов отчетов, выполняющих схожий анализ. Выберите из выпадающего списка нужный вариант отчета.

Настройка строк

  • настройка строк отчета выполняется в разделе Строки.
    • флажками можно управлять видимостью группировок строк
    • добавление новой группировки строк выполняется по кнопке + Добавить строку.
    • управление порядком группировок строк выполняется по кнопке , затем с помощью стрелок  и

Настройка колонок

  • Настройка колонок выполняется в разделе Колонки.
    • флажками можно управлять видимостью колонок.
    • добавление новой колонки выполняется по кнопке + Добавить колонку.
    • управление порядком колонок выполняется по кнопке , затем с помощью стрелок  и
    • выпадающий список Сгруппировать по позволяет группировать колонки по указанному полю отчета
    • выпадающий список Сравнить по становится доступным, когда отмечена только одна колонка, при этом:
      • флажок +/- включает отображение разницы в абсолютных величинах
      • флажок % включает отображение разницы в относительных величинах

Сохранение настроек отчета

  • Нажмите Запомнить настройки, укажите Наименование для сохраненных настроек и другие параметры.
  • Нажмите Еще - Прочее - Стандартные настройки, для того чтобы сбросить пользовательские настройки.

Другие доступные действия

  • Гиперссылка свернуть управляет видимостью панели настроек.
