Отчет "Сводный анализ заказов поставщикам"

Отчет предназначен для анализа состояния заказов поставщикам в части оплаты и поступления.

В части оплаты товаров по заказу выводятся следующие показатели:

  • Сумма документа – сумма запланированной оплаты, то есть та сумма, которая указана в заказе поставщику.
  • Оплачено - сумма оплаченная по данному заказу.
  • Осталось оплатить – сумма, которую необходимо погасить, чтобы полностью оплатить заказ.

В части поступления по заказу выводится количество товаров в базовых единицах измерения:

  • Заказано – количество товаров, которое указано в заказе.
  • Поступило - количество товаров, которое поступило по заказу.
  • Осталось закупить - общее количество товаров, которое осталось получить по заказу.
  • Размещено в заказах - количество товаров по заказу покупателя, которое размещено в данном заказе поставщику.
